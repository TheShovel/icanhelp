# General Reference Knowledge

## Time & Calendar

### Time Zones
```
UTC = Coordinated Universal Time (Zulu)
EST/EDT = UTC-5/-4 (US East)
CST/CDT = UTC-6/-5 (US Central)
MST/MDT = UTC-7/-6 (US Mountain)
PST/PDT = UTC-8/-7 (US West)

Common offsets:
London: UTC+0/+1 (GMT/BST)
Paris/Berlin: UTC+1/+2 (CET/CEST)
Moscow: UTC+3
Dubai: UTC+4
Mumbai: UTC+5:30
Shanghai/Singapore: UTC+8
Tokyo: UTC+9
Sydney: UTC+10/+11 (AEST/AEDT)
```

### Date Formats
```
ISO 8601: YYYY-MM-DD (2024-01-15) - unambiguous
US: MM/DD/YYYY (01/15/2024)
UK/EU: DD/MM/YYYY (15/01/2024)
Unix timestamp: Seconds since 1970-01-01 00:00:00 UTC
```

### Leap Years
```
Divisible by 4, except centuries not divisible by 400
2000: leap (÷400)
1900: not leap (÷100, not ÷400)
2024: leap
```

## Common Formulas

### Geometry
```
Circle:  C = 2πr, A = πr²
Sphere:  V = ⁴/₃πr³, A = 4πr²
Cylinder: V = πr²h, A = 2πr(r+h)
Cone:     V = ⅓πr²h
Triangle: A = ½bh, a² + b² = c² (right)
Rectangle: A = lw, P = 2(l+w)
Trapezoid: A = ½(a+b)h
```

### Trigonometry
```
sin θ = opposite/hypotenuse
cos θ = adjacent/hypotenuse
tan θ = opposite/adjacent
sin²θ + cos²θ = 1
Law of sines: a/sin A = b/sin B = c/sin C
Law of cosines: c² = a² + b² - 2ab cos C
```

### Algebra
```
Quadratic: ax² + bx + c = 0 → x = (-b ± √(b²-4ac))/2a
Distance: d = √((x₂-x₁)² + (y₂-y₁)²)
Midpoint: ((x₁+x₂)/2, (y₁+y₂)/2)
Slope: m = (y₂-y₁)/(x₂-x₁)
Line: y = mx + b or y - y₁ = m(x - x₁)
```

### Finance
```
Simple interest: I = Prt
Compound: A = P(1 + r/n)^(nt)
Continuous: A = Pe^(rt)
APY = (1 + r/n)^n - 1
Rule of 72: Years to double ≈ 72/rate%
CAGR = (End/Start)^(1/years) - 1
NPV = Σ CFₜ/(1+r)ᵗ
```

### Statistics
```
Mean: x̄ = Σx/n
Median: middle (or avg of two middle)
Mode: most frequent
Range: max - min
Variance: σ² = Σ(x-μ)²/n
Std dev: σ = √variance
Z-score: z = (x-μ)/σ
Standard error: SE = σ/√n
CI: x̄ ± z·SE
```

## Programming Patterns

### Regex Basics
```
.       Any char except newline
\d      Digit [0-9]
\D      Non-digit
\w      Word char [a-zA-Z0-9_]
\W      Non-word
\s      Whitespace
\S      Non-whitespace
^       Start of string/line
$       End of string/line
*       0 or more
+       1 or more
?       0 or 1
{n,m}   n to m times
[abc]   a, b, or c
[^abc]  Not a, b, or c
(a|b)   a or b
(...)   Capture group
(?:...) Non-capture group
```

### Design Patterns (Gang of Four)
```
Creational:
  Singleton: One instance globally
  Factory: Create objects without specifying class
  Builder: Step-by-step construction
  Prototype: Clone existing objects

Structural:
  Adapter: Interface conversion
  Decorator: Add behavior dynamically
  Facade: Simplified interface
  Proxy: Placeholder/control access

Behavioral:
  Observer: Notify dependents of changes
  Strategy: Interchangeable algorithms
  Command: Encapsulate request as object
  Iterator: Sequential access
  Template Method: Algorithm skeleton
```
