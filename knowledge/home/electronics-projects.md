# Electronics & Home Wiring Projects

## Basic Electronics

### Components
- **Resistors**: limit current. Color code: black 0, brown 1, red 2, orange 3, yellow 4, green 5, blue 6, violet 7, gray 8, white 9. 4 bands: digit-digit-multiplier-tolerance (gold=5%, silver=10%). 5 bands: digit-digit-digit-multiplier-tolerance (for precision). Series: R_total = R1+R2+... Parallel: 1/R_total = 1/R1 + 1/R2 + ...
  - Power rating: 1/8W (small, common for signals), 1/4W (most hobby projects), 1/2W, 1W+ (for power circuits). Use resistor wattage 2x actual dissipation
  - Variable resistor = potentiometer (pot): adjust resistance. Trimpots: small, for calibration. Pots used as volume controls, voltage dividers. Potentiometer with three terminals: adjust wiper between fixed ends
- **Capacitors**: store charge. Units: farad (F — usually micro µF, nano nF, pico pF). Types: ceramic (small values, non-polarized, cheap. Common: 0.1µF bypass). Electrolytic (large values 1+µF, polarized — one side marked with stripe -. Used for power supply smoothing, audio coupling. Failure mode: explode if polarity reversed, leak or dry out over years). Tantalum (medium values, polarized, stable, prone to catastrophic failure if voltage exceeded). Film (polyester/polypropylene: non-polarized, stable, good for audio/timing). Supercapacitors: huge (F range, low voltage, slow discharge, for backup power/ RTC)
  - Capacitor equation: Q = C×V. RC time constant: τ = RC, charges to ~63% in one τ, 5τ to fully charged. Used for timing circuits, filters, and smoothing (power supply ripple)
- **Diodes**: conduct one direction only. Standard (1N4001-1N4007 — "rectifier" for AC→DC power. Rating: 1A, 50-1000V PIV). Fast recovery/schottky (low forward voltage, fast switching, appropriate for high-frequency switching supplies). Zener (reverse breakdown at specific Zener voltage — used as voltage reference or shunt regulator: breaks down at a designed voltage). LED (light emitting diode — forward voltage ~2V, current ~20mA. Always limit current with resistor (R = (Vsupply - Vled) / 20mA). Blue/white LEDs ~3V, red ~2V)
- **Transistors**: BJT (bipolar junction transistor) — current-controlled. NPN (switches on when base current applied: 0.7V base-emitter voltage). PNP (switches when base pulled LOW). Uses: switch (digital on/off), amplifier (analog)
  - MOSFET (metal-oxide-semiconductor field-effect transistor) — voltage-controlled gate. No gate current (high input impedance). P-channel vs N-channel. N-channel: gate HIGH > source voltage turns on. R_DS(on): resistance when fully on. IRF520/IRF540: common N-channel, moderate current. SMPS: high-frequency, high-speed switching with low losses. MOSFET gate threshold: Vgs(th) = 2-4V typical. Use gate resistor + zener clamp to prevent oscillation/damage
- **ICs**: operational amplifiers (op-amps: LM358, TL072, NE5532 — analog signal processing: amplify, buffer, filter. Virtual ground concept). 555 timer (astable = oscillator, monostable = pulse, bistable = latch. Most popular IC ever). Voltage regulators (7805 = +5V out 1A, 7812/+12V, adjustable LM317. Need input/output caps). Logic gates (74HC series: AND, OR, NOT, NAND, NOR, XOR. D flip-flip, counter — 4000 CMOS series also). Arduino (microcontroller — ATmega328P. 5V logic, 14 digital I/O + 6 analog input, 16 MHz. 32KB flash memory for sketches). Raspberry Pi (single board computer — Linux OS, general purpose computer GPIO, up to 40 pins. HDMI out, camera, USB. For higher-level control: Python, not real-time. Good for IoT gateway — not for time-critical/motor control. Run Linux OS)
- **Sensors**: temperature (thermistor — resistance changes with temperature. Resistance: 10k at 25C typical NTC; LM35 = 10mV/°C; DHT11/DHT22 = temperature + humidity digital output). Light (LDR/CDS photoresistor, phototransistor, photodiode). Motion (PIR — passive infrared, detects body heat, HC-SR501 common). Distance/sonar (HC-SR04 ultrasound: trigger pulse → measure echo pin pulse width → divide by 58 = cm). Accelerometer (MPU6050 — 3-axis accel + gyro, I2C). Current sensor (ACS712 — hall effect, measures AC+DC current, isolated, output 185mV/A for 5A version)

## Breadboard Prototyping
- **Rows**: connected horizontally (each numbered row, 5 holes each, connected inside). Power rails: vertically along sides (+) (−) usually red/blue
  - Internal connections: rows are connected (split in middle in center — each row has two halves). Power rails on each side are continuous vertically. Check continuity with multimeter
- **Jumpers**: 22 AWG solid wire (not stranded). Pre-formed jumper wire kit. Dupont wires for Arduino/ Raspberry Pi connections
- **Schematic → breadboard**: plan connections — don't rely on memory. Check: no floating inputs, polarized components (electrolytic caps, diodes, IC orientation/pin1 notch). Add decoupling capacitor (0.1µF) near IC pins (especially digital ICs — bypass noise). Never power up without checking for short between VCC and GND
- **Multimeter**: continuity (beep = connection). Voltage (measure in parallel, across component). Resistance (measure with power OFF). Current (measure in series, break circuit). Diode test: voltage drop across junction (0.6-0.7 for silicon, 0.2 for Schottky). Capacitance (some meters)

## Soldering Techniques
- **Through-hole**: heat pad + lead simultaneously (1-2 sec), apply solder to the joint (not on iron tip). Solder flows = good joint (shiny cone shape). Remove iron, let cool (5 sec). Good: conical, shiny, smooth, volcano-like shape, component lead visible through solder. Bad: cold joint (dull, granular, cracked) = poor electrical connection + failure over time. Ball joint (too much solder, can touch adjacent). Bridge (accidental connection to adjacent). Disturbed joint: moved while cooling — crystalline
  - Desolder: wick (braid) — place over joint, heat with iron → wick absorbs molten solder. Solder sucker (squeeze, melt, release button = vacuum). Heat gun on low, for SMD removal. Chip Quik: low melt alloy + heat, remove component
- **SMD (surface mount)**: smaller, no holes. Hand soldering: 0603 (0.8mm) and larger doable with tweezers + fine iron tip. 0402 (0.5mm) very hard, 0201 near impossible by hand. Hot air rework: solder paste with stencil → reflow at temperature curve for solder type. Infrared reflow. For hobby hand-soldering: use flux pen on pads, melt solder on one pad, position component, reheat pad to tack a leg, solder remaining pins. Use flux: never skip. Use brass sponge to clean iron tip

## Home Wiring Projects (Practical)

### Installing a Ceiling Fan
- **Pre-existing light fixture**: turn off breaker, test with non-contact voltage tester (NCVT). Remove old fixture. Check box: rated for ceiling fan (marked "For fan support" or "7/8" hole for box rated 70 lbs)? If standard box, need fan-rated box (bracket through into joist). Wire connections: black (fan light hot), blue (fan motor hot), white (neutral), green/green-yellow (ground). Connect: fan black + blue to house black (hot). Fan white to house white (neutral). Grounds together. Secure fan to mounting bracket. Balance blades (included kit: add weights if wobbling). Install light kit + fan blades

### Installing a Dimmer Switch
- Smart dimmers: need neutral wire (white). Standard dimmers: 2 wires (line + load, plus ground). Replace switch (single pole). If neutral required for smart: check box for white wire bundle. If none (older switch loops): need smart dimmer that works without neutral (Caséta, Lutron, Inovelli dimmers) or run a neutral if needed. Verify LED dimmer compatible with your bulbs (LED bulb needs "dimmable" rating). Match wattage: dimmer rating >= maximum bulb wattage on the switch. LEDs reduce total load but can cause flicker if load is low — check dimmer minimum

### Installing a GFCI Outlet
- **Where required**: kitchen countertops (all within 6 ft of sink), bathrooms (all outlets), garages, basements (unfinished), outdoors, crawlspaces, near laundry/utility sinks, within 6 ft of any sink. GFCI works: detects imbalance in current between hot + neutral (if leak to ground). Trip threshold: ~5mA within 1/40 sec
  - Install: identify "LINE" (power from panel — marked) and "LOAD" (to downstream outlets) terminals. Line: incoming hot + neutral. Load: if you want downstream outlets also protected. Connect wires to correct screws. Label: "GFCI Protected" if you use downstream protection through load side. Test: press TEST → should click off, RESET to restore. Monthly test recommended
- If box has no ground wire: install GFCI and mark "No Equipment Ground." GFCI will still protect against shock, but 3-prong outlets downstream will not have ground — code still allows GFCI on ungrounded circuit and labels "GFCI Protected / No Equipment Ground"

### Running a New Circuit
- **Significant work**: usually requires permit. Breaker in panel → wire to first box → daisy chain to more outlets. Wire size: 14 AWG for 15A circuit, 12 AWG for 20A — larger gauge for higher amperage. Use NM-B cable (type Romex). Drill holes in center of studs (not within 1.25" of edge). Secure cable with staples within 12" of each box, 4-6 feet between. Leave ~6" wire tail at each box. Label circuits at panel

## Electrical Troubleshooting
- **Breaker trips repeatedly**: unplug everything on circuit. If still trips: short circuit (hot touching neutral or ground). Call electrician. If stops tripping: plug items back one by one — find which appliance is faulty
  - Intermittent trip: overloading during high load (microwave + toaster + air fryer on same 15A circuit = too many amps)
- **GFCI won't reset**: miswired (line/load reversed), doesn't have power (tripped upstream GFCI on the same circuit), bad GFCI (replace if Test/Reset fails multiple correct checks)
- **Lights flicker**: loose bulb (tighten), bad bulb (replace). Loose wire connection in fixture. Dimmer/LED mismatch. If whole house flickers: large appliance starting (well pump, AC) on startup. If consistent: main connection issue at service panel
- **Outlet not working**: check breaker (on?). Check all GFCI outlets upstream (press RESET). Test outlet with multimeter. If no power: loose wire at outlet or at previous outlet in chain (backstabbed wires fail: break the old backstab terminal, side-screw new wires). Tripped GFCI on same circuit, not the same room
- **Dead outlet with voltage tester beeping**: loose neutral wire (neutral carries return current and is not ground). NCVT beeps near hot, won't tell you about neutral. If outlet has power but device won't work = open neutral (white wire broken somewhere). Check previous outlet in chain

## Advanced Home Automation
- **Hub**: Zigbee/Z-Wave hub (Hubitat, Home Assistant on Raspberry Pi, SmartThings). vs WiFi direct (no hub required). Z-Wave: mesh, uses 908 MHz (US) — less interference than 2.4 GHz. Zigbee: 2.4 GHz mesh, more interference from WiFi. Home Assistant = most powerful, open-source, steep learning curve, large ecosystem
  - Best: Z-Wave range + reliability. Zigbee mesh for ZLL/ZHA lighting. WiFi for simple bulbs/switches (security concerns depend on brand — stick to trusted brands: Leviton, Lutron Caséta, GE/Jasco, Zooz)
- **Smart switches (> smart bulbs)**: work with existing bulbs, don't need "smart bulb mode." Neutral wire required for most. Lutron Caséta (expensive but rock solid, no neutral required, Pico remote = any-place switch position). Inovelli (Z-Wave, multi-function from single switch — notifications, scenes). Zooz (Z-Wave, affordable). Shelly (WiFi relay behind existing switch — very small, modular, for in-wall/box use with phone/DIY)
- **Automation**: schedules (lights on/off). Scenes (multiple devices). Rules: "if motion detected AND sunset → turn on lights 30%" (condition builder). Sensor input + switches + relays. Dummy switch = virtual state — use for integration across systems
