# Home Maintenance Schedule

## Overview
Preventive maintenance protects your home's value, prevents costly repairs, and ensures safety. This schedule covers monthly, seasonal, and annual tasks.

## Monthly Tasks

### Safety Systems
- [ ] Test smoke detectors (press test button)
- [ ] Test CO detectors (press test button)
- [ ] Check fire extinguisher pressure gauge
- [ ] Test GFCI outlets (press test/reset)
- [ ] Check AFCI breakers (test button)

### HVAC
- [ ] Check/change air filter (1-3" = monthly, 4-5" = quarterly)
- [ ] Clear debris from outdoor unit (2 ft clearance)
- [ ] Check thermostat operation

### Plumbing
- [ ] Run water in unused fixtures (guest bath, laundry sink)
- [ ] Check under sinks for leaks
- [ ] Test sump pump (pour water in pit)
- [ ] Check water softener salt level

### Exterior
- [ ] Clear gutters/downspouts (if trees nearby)
- [ ] Check for foundation cracks
- [ ] Inspect weatherstripping on doors/windows

### Interior
- [ ] Clean range hood filter
- [ ] Clean garbage disposal (ice + citrus)
- [ ] Check caulk/tub surround for gaps
- [ ] Test garage door auto-reverse

## Quarterly (Every 3 Months)

### HVAC
- [ ] Clean condenser coils (outdoor unit)
- [ ] Check refrigerant lines for insulation damage
- [ ] Verify condensate drain flows (pour vinegar/water)
- [ ] Check ductwork for disconnections/leaks

### Plumbing
- [ ] Flush water heater (drain 2-3 gallons)
- [ ] Test pressure relief valve (lift lever)
- [ ] Check anode rod (every 2-3 years, replace if <1/2" thick)
- [ ] Clean faucet aerators/showerheads (vinegar soak)

### Electrical
- [ ] Test all outlets (receptacle tester)
- [ ] Check breaker panel for warmth/discoloration
- [ ] Tighten loose outlet/switch plates
- [ ] Check extension cords for damage

### Appliances
- [ ] Clean refrigerator coils (vacuum brush)
- [ ] Check dishwasher filter
- [ ] Clean washing machine (vinegar cycle, wipe gasket)
- [ ] Clean dryer vent (inside + outside)

### Exterior
- [ ] Inspect roof from ground (binoculars)
- [ ] Check grading slopes away from foundation
- [ ] Inspect siding/trim for damage
- [ ] Check deck/patio for loose boards, rot
- [ ] Lubricate garage door springs/rollers

## Seasonal

### Spring (March-May)
```
☐ HVAC professional tune-up (AC focus)
☐ Clean gutters & downspouts thoroughly
☐ Inspect roof (shingles, flashing, vents)
☐ Check attic for leaks, mold, insulation
☐ Service sprinkler system (backflow test)
☐ Power wash siding, deck, driveway
☐ Repair caulk/paint exterior as needed
☐ Check window/door screens
☐ Fertilize lawn, aerate if needed
☐ Prune trees/shrubs away from house
☐ Test well water (if applicable)
☐ Check septic system (if applicable)
```

### Summer (June-August)
```
☐ Monitor AC performance (temp differential 15-20°F)
☐ Check condensate drain weekly
☐ Mow at proper height (3-4" for cool season)
☐ Water deeply, infrequently (1-1.5"/week)
☐ Inspect for pests (ants, termites, wasps)
☐ Clean grill, check propane
☐ Check pool/spa chemistry weekly
☐ Trim vegetation from AC unit
☐ Check for irrigation leaks
```

### Fall (September-November)
```
☐ HVAC professional tune-up (furnace focus)
☐ Clean gutters (after leaves fall)
☐ Drain/blow out sprinkler system
☐ Shut off exterior faucets, drain hoses
☐ Insulate exposed pipes
☐ Check weatherstripping, add storm windows
☐ Clean chimney/flue (if wood burning)
☐ Test sump pump before wet season
☐ Rake leaves, mulch or compost
☐ Fertilize lawn (cool season grasses)
☐ Store outdoor furniture/cushions
☐ Check attic insulation depth
```

### Winter (December-February)
```
☐ Monitor for ice dams (attic ventilation)
☐ Check for drafts (candle test)
☐ Keep vents clear of snow/ice
☐ Run water in unused fixtures weekly
☐ Monitor humidity (30-50% indoor)
☐ Check roof after heavy snow (rake if needed)
☐ Test backup generator monthly
☐ Keep walkways clear, use safe ice melt
☐ Check for frozen pipes on cold nights
```

## Annual Tasks

### Professional Services
- [ ] HVAC full inspection & cleaning
- [ ] Chimney sweep & inspection (if wood/gas fireplace)
- [ ] Termite/pest inspection
- [ ] Septic tank inspection/pumping (3-5 years)
- [ ] Well water test (bacteria, nitrates, arsenic)
- [ ] Electrical panel inspection (every 5-10 years)
- [ ] Roof professional inspection (after 10 years)

### Deep Cleaning
- [ ] Clean refrigerator interior (remove shelves)
- [ ] Clean oven (self-clean or manual)
- [ ] Wash windows inside & out
- [ ] Deep clean carpets/rugs
- [ ] Clean upholstery
- [ ] Organize garage/shed/attic
- [ ] Purge expired medications, chemicals, food

### Systems
- [ ] Flush water heater fully
- [ ] Replace water filter cartridges
- [ ] Check/replace anode rod
- [ ] Test backup sump pump battery
- [ ] Inspect foundation for new cracks
- [ ] Check retaining walls for movement
- [ ] Lubricate all door hinges, locks
- [ ] Check attic ventilation (baffles clear)
- [ ] Inspect crawlspace/vapor barrier

## By Component Lifecycle

### Replace Every 1-2 Years
- HVAC filters (1-3" monthly, 4-5" quarterly)
- Refrigerator water filter
- Range hood filter (clean monthly, replace annually)
- Humidifier pad/water panel
- UV light bulb (if installed)
- Carbon monoxide detector (5-7 years)

### Replace Every 3-5 Years
- Smoke detectors (10 years max)
- CO detectors (5-7 years)
- Water heater anode rod
- Washer hoses (stainless braided)
- Toilet flappers/fill valves
- Faucet cartridges
- Garage door rollers/springs
- Sump pump (5-7 years)

### Replace Every 10-15 Years
- Water heater (tank: 8-12, tankless: 15-20)
- HVAC system (AC: 10-15, Furnace: 15-20, Heat pump: 10-15)
- Roof (asphalt: 20-30, metal: 40-70, tile: 50+)
- Windows (20-30 years)
- Siding (vinyl: 20-40, fiber cement: 50+)
- Electrical panel (30-40 years)
- Plumbing pipes (copper: 50+, PEX: 40-50, PVC: 50+)

### Replace Every 20-30 Years
- Kitchen appliances (fridge: 13, dishwasher: 9, range: 13-15)
- Washer/dryer (10-13 years)
- Garage door opener (10-15 years)
- Deck (wood: 10-15, composite: 25-30)
- Driveway (asphalt: 15-20, concrete: 25-30)

|------------|---------------|---------|

| $300k | $3,000-12,000 | $250-1,000 |
| $400k | $4,000-16,000 | $333-1,333 |

**1% rule**: Basic maintenance
**2-3% rule**: Older home, deferred maintenance
**4% rule**: Major systems nearing end of life
