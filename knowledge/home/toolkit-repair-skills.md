# Home Toolkit & Core Repair Skills

## Overview
Every home needs a basic toolkit and the skills to handle common repairs. This guide covers essential tools, fundamental skills, and step-by-step fixes for the most common household issues.

## Fundamental Skills

### 1. Finding Studs
```
Methods (in order of reliability):
1. Stud finder (calibrate on empty wall)
2. Electrical outlets/switches (always on stud side)
3. Baseboard nails (visible or feel)
4. Measure from corner (16" or 24" OC)
4. Knock test (hollow vs solid sound)
5. Strong magnet (finds drywall screws)

Verify: Drill small pilot hole, feel for wood resistance
```

### 2. Hanging Things Securely
```
< 10 lbs: Picture hook, adhesive strip
10-25 lbs: Picture hook into stud, or drywall anchor (toggle, molly)
25-50 lbs: Screw into stud, or heavy-duty toggle (Snaptoggle)
> 50 lbs: Must hit stud, or lag bolt into blocking

Drywall anchor types:
- Plastic expansion: < 10 lbs, light duty
- Self-drilling (ZIP-IT): 25-50 lbs, easy install
- Toggle bolt: 50-100+ lbs, needs 1/2" hole
- Snaptoggle: 100-200+ lbs, best for heavy items
```

### 3. Basic Drywall Repair
```
Small hole (< 1/2"):
1. Clean edges
2. Spackle, overfill slightly
2. Dry, sand, repeat if needed
4. Prime, paint

Medium hole (1/2" - 4"):
1. Cut square around damage
2. Cut patch piece same size
3. Backing: wood strips behind, screwed to studs
4. Screw patch to backing
5. Mesh tape joints
5. 3 coats joint compound, feathering
6. Sand, prime, paint

Large hole (> 4"):
- Cut to stud centers
- Add backing between studs
- Patch full sheet section
```

### 4. Caulking (Pro Results)
```
Prep:
- Remove ALL old caulk (caulk remover, razor, tool)
- Clean: alcohol, dry completely
- Tape both sides for clean line

Apply:
- Cut tip at 45°, bead size = gap width
- Steady pressure, smooth continuous motion
- Push caulk INTO gap, don't just lay on top

Tool:
- Wet finger or caulk tool
- Smooth in ONE pass
- Remove tape immediately
- Clean up with damp rag

Types:
- Siliconized acrylic: Paintable, interior, 15-20 yr
- 100% Silicone: Wet areas, not paintable, 20+ yr
- Hybrid (STP): Paintable + waterproof, best all-around
- Polyurethane: Exterior, high movement, paintable
```

### 5. Basic Plumbing
```
Shutoff valves: Know locations!
- Main: Usually near meter, street side
- Fixture: Under sinks, behind toilets
- Water heater: Cold inlet valve

Leaky faucet (compression):
1. Shut off water (fixture or main)
2. Remove handle, packing nut
3. Replace washer/O-ring
4. Reassemble, test

Running toilet:
1. Check flapper (most common) - replace $5
2. Check fill valve - adjust or replace
3. Check chain length - slight slack only

Clogged drain:
1. Plunger: Flange type, seal overflow
2. Snake/auger: 1/4" for sinks, 3/8" for toilets
3. Chemical: Avoid (damages pipes, dangerous)
4. Cleanout: Access point for main line

Supply line replacement:
- Flexible braided stainless: 10-15 yr life
- Replace at fixture changes
- Don't overtighten: hand tight + 1/4 turn
```

### 6. Basic Electrical (Safety First!)
```
RULES:
- ALWAYS test for power (non-contact tester)
- Turn off breaker, verify dead
- One hand in pocket (heart path protection)
- Permit/inspection for new circuits

Outlet replacement:
1. Power off, verify
2. Remove old, note wire positions
3. Black/red → brass (hot)
4. White → silver (neutral)
5. Green/bare → green (ground)
6. Secure, test, cover

Switch replacement:
- Same as outlet but brass terminals only
- Ground to green
- Note: white may be hot (mark with black tape)

GFCI:
- Required: Kitchen, bath, garage, outdoor, laundry
- Test monthly (test/reset buttons)
- Protects downstream outlets too

Light fixture:
- Box rated for weight (fan = listed box)
- Black→black, white→white, ground→ground
- Support heavy fixtures independently
```

### 6. Appliance Basics

#### Refrigerator
- **Coils**: Clean annually (brush + vacuum)
- **Door seals**: Test with dollar bill, clean gasket
- **Temp**: 37°F fridge, 0°F freezer
- **Level**: Front slightly high for door seal

#### Dishwasher
- **Filter**: Clean monthly (bottom)
- **Spray arms**: Clear holes annually
- **Drain**: Check air gap/loop
- **Don't**: Pre-rinse (modern need food for enzymes)

#### Washer/Dryer
- **Washer**: Clean filter (front load), leave door open
- **Dryer**: Lint trap every load, vent annually
- **Hoses**: Replace rubber with braided stainless (10 yr)
- **Level**: Adjust feet, prevents vibration

#### HVAC
- **Filter**: MERV 8-11, change 1-3 months
- **Condenser**: Clear 24", clean fins annually
- **Drain line**: Bleach/vinegar flush quarterly
- **Pro tune-up**: Annual (spring AC, fall heat)

## Common Repair Recipes

### Stuck Window/Door
```
1. Clean tracks (vacuum, brush)
2. Lubricate: Silicone spray or dry PTFE
3. Check alignment: Shims, hinge adjustment
4. Paint bind: Score, free with putty knife
5. Sash cords: Replace if broken (sash weights)
```

### Squeaky Floor/Stairs
```
From below: Shim between joist/subfloor
From above: Screw through subfloor into joist
Special: Squeak-No-More kit (break-away screws)
Lubricate: Talc/graphite between boards
```

### Running Toilet (Detailed)
```
1. Flapper: Replace (universal $5)
   - Check chain: 1/2" slack
2. Fill valve: Adjust float, or replace (Fluidmaster 400A)
3. Overflow tube: Water 1" below top
4. Tank bolts: Tighten if leaking at base
```

### Garbage Disposal Jam
```
1. Power OFF (unplug or breaker)
2. Reset button (bottom)
3. Allen wrench (bottom) - turn back/forth
4. Tongue/needle nose pliers - remove object
5. Run cold water, test
```

### GFCI Won't Reset
```
1. Unplug everything on circuit
2. Press RESET firmly
3. If holds: Plug items one by one (find fault)
4. If won't hold: Replace GFCI ($15)
```

## When to Call a Pro
- **Electrical**: New circuits, panel work, aluminum wiring
- **Gas**: Any gas line work
- **Structural**: Load-bearing walls, foundation cracks > 1/4"
- **Roof**: Steep, high, or major repairs
- **Permit required**: Structural, electrical, plumbing, HVAC
- **Beyond skill/tools**: Risk of damage > pro cost
- **Time-sensitive**: Active leaks, no heat/AC in extreme weather

## Finding Good Contractors
- **Referrals**: Friends, neighbors, hardware store staff
- **Reviews**: Google, Angi, Nextdoor (filter for recent, similar work)
- **Licenses**: Verify state license + insurance
- **Estimates**: 3 minimum, detailed scope
- **Payment**: Never > 10% down, milestone payments, final on completion
- **Contract**: Written, detailed, timeline, warranty, lien waiver

## Building Your Skills
- **YouTube**: This Old House, Essential Craftsman, Home Repair Tutor
- **Books**: "Complete Do-It-Yourself Manual" (Family Handyman), "Dare to Repair"
- **Classes**: Home Depot/Lowes workshops, community college, makerspace
- **Practice**: Start small, expect mistakes, learn each time
- **Document**: Photos before/during/after, receipts, notes for next time

## Safety Non-Negotiables
- [ ] Safety glasses: ALWAYS
- [ ] Hearing protection: Power tools
- [ ] Respirator: Dust, chemicals, insulation
- [ ] Gloves: Appropriate to task
- [ ] Ladder safety: 4:1 angle, 3-point contact, no top 2 rungs
- [ ] Electrical: Test before touch, one hand
- [ ] Lifting: Bend knees, keep load close, get help
- [ ] Chemical: Ventilation, no mixing, read SDS
- [ ] Fire extinguisher: Accessible, charged, know PASS
- [ ] First aid kit: Stocked, accessible
