# Home Security & Safety

## Doors & Entry Points
- **Exterior doors**: solid wood or metal (hollow core = kicks open easily). Door should fit frame tightly
  - Hinges: inside only (exterior hinge = pull pin = door comes off. If exterior hinges: security hinge pins work)
  - Peephole: wide-angle, 180° preferred (digital peephole with camera available)
- **Deadbolt**: Grade 1 (ANSI commercial grade = strongest, most burglars can't kick). Single-cylinder (key outside, thumb turn inside). Double-cylinder (key both sides — needed if glass near lock so burglar can't reach through and unlock — but fire hazard if no key inside = code issue)
  - Strike plate: reinforce with 3" screws into wall stud (common weak point — long screws through into framing dramatically increases kick-in resistance)
- **Smart lock**: keyless entry, codes for specific people (temporary codes for contractor/cleaner), remote lock/unlock, auto-lock timer, activity log — but vulnerable to hacking (choose Z-Wave/Zigbee/Thread, use strong WiFi, firmware updates, 2FA)
  - Brand: Schlage, August, Yale, Level (hidden lock). Locks should be BHMA/ANSI certified
- **Sliding glass door**: dowel in track ($0 — strongest cheap fix), security pin (through frame, prevents lifing out of track), security film (prevents easy break), glass break sensor (alarm)
  - Shims: set door track so it can't be lifted out. Most sliding doors can be lifted out of frame — shimming prevents
- **Garage door**: never leave opener visible in car (break car window = house key — valet parking duplicate risk). Disconnect emergency release zip-tie (break stick through release cord = can open door manually). Secure service door
  - Garage-to-house door: treat as EXTERIOR DOOR (solid, deadbolt, reinforced) — common weak entry point
  - Smart garage opener: MyQ, alerts when left open, close remotely

## Windows
- **Locks**: basic locks prevent casual entry — better: pin (drill hole through overlapping sash frames, insert removable nail). Keyed window lock (needs key to open — fire hazard if emergency, keep key accessible)
  - Double-hung: security pin through inner sash into frame. Casement: crank lock (harder to force than sliding sash)
- **Security film**: 3M safety/security film — makes window harder to break through (rage-breaker: kicks window 4-5 times before through instead of once). Not bulletproof
  - 8mil-12mil thickness for security (thicker = more protection, harder to install). DIY vs professional install
- **Bars/gates**: effective but fire hazard (must have quick-release mechanism). Check local fire code
- **First floor**: thorny bushes under windows (barberry, rose, holly, pyracantha). Decorative gravel (crunchy)

## Alarm Systems
- **Professional** (ADT, Vivint, SimpliSafe): 24/7 monitoring, police dispatch, monthly fee ($20-50/month), contract (1-3 years typical). Professional install or self-install options
  - SimpliSafe = easy DIY, no contract. Ring Alarm = cheap ($10/mo monitoring, Amazon integration). ADT = most established, contracts, most expensive
- **Self-monitored**: Ring, Wyze, Abode, Aqara — app alerts your phone, you call police. No monthly fee (or minimal for cloud recording). Good for: people who want alerts but not monthly cost
- **Sensors**: door/window (magnetic = open/close), motion (PIR — pets ≤ 40 lbs can be filtered out off-limit. Placement: corner of room, 7-8 ft up, NOT facing window/sun/heat vent), glass break (listens for specific frequency of breaking glass), smoke/CO (interconnected = each triggers all), flood/water leak (near water heater, washer, under sinks, sump pump), freeze sensor (30°F alert — pipes)
  - Door sensor placement: aligned on door + frame, gap 1/4" or less. Can install on garage door, mailbox, medicine cabinet

## Cameras
- **Indoor**: good for checking pets/kids/caregivers. Privacy concerns (hackable — choose reputable brand + 2FA + regular firmware updates, point away from private areas if not needed)
  - Placement: common area, main entry, pet areas. Privacy: turn off or cover when home if preferred
- **Outdoor**: visible = deterrent (but can be stolen/damaged), hidden/less obvious = evidence but won't prevent. 1080p+ resolution, night vision (color night vision with spotlights), weatherproof (IP65+)
  - Placement: front door (most common entry), back door, garage, side gates, driveway (capture car/license plates). 8-10 ft up, angled slightly down (prevents vandalism, wider field of view)
- **Doorbell camera**: Ring, Nest, Eufy, Logitech — see who's at door, 2-way audio, package theft monitoring
  - Subscription: cloud recording usually $3-10/month (important — without it you see live but can't review). Eufy = no subscription needed (local storage)
- **Storage**: cloud (monthly fee, accessible from anywhere, evidence safe if camera stolen) vs local (SD card/NVR — no fee, camera destroyed = video lost if recording on device. Hard drive NVR/computer better. Key: have both)

## Lighting & Deterrence
- **Motion lights**: floodlights/motion-sensor lights at entry points, driveway, back yard — bright LED (3000+ lumens). Surprise + visibility = strong deterrent
  - Position: over each exterior door, garage, dark corners, path to doors. 10-12ft high, angled down + out
- **Schedule lights**: interior lights on timer when away — TV simulator (randomizes like real usage). Smart bulbs (Philips Hue) with away mode
- **Landscaping**: trim bushes below window height (remove hiding spots). Thorny plants under windows. Gravel paths near entry points (crunchy = audible)
- **Signs + stickers**: alarm company sign in yard + window stickers — simple deterrent (even if no alarm). "Beware of dog" sign (even without dog). Fake cameras: obvious fakes don't work (burglars recognize them), realistic ones with blinking light better

## While Away
- **Mail**: hold at post office or neighbor picks up. Overflowing mailbox = vacancy signal
- **Car**: move car occasionally (or have neighbor park in your spot/driveway). If you leave car parked: have neighbor move it
- **Social media**: DON'T post travel plans in advance. Post AFTER return. Wait to post vacation photos until home
- **Timer lights + radio**: random timers (not same time every day). TV simulator (FakeTV $25-40 mimics TV glow, more convincing than lamp timer)
- **Trash cans**: don't leave them out on street (if away pickup day, neighbor puts out + brings in)
- **Check-in**: trusted neighbor has key (emergency contacts, brings in packages, alternate parking). Leave itinerary with someone

## Digital Security Related to Home
- **WiFi security**: WPA3 (or WPA2 at minimum — WEP = easily cracked). Strong password (not "password" or "Linksys"). Hide SSID optional (security through obscurity = minimal benefit, increases nuisance)
  - Guest network for IoT devices/smart home (compromised smart bulb can't access your PC)
  - Change admin password on ALL smart devices (default passwords = searchable online)
- **Smart home security**: regular firmware updates, strong unique password per device, disable remote access if not needed, consider separate VLAN for IoT (isolate from computer/phone)
  - Biggest risk: smart locks + cameras hacked → privacy invasion + entry. Mitigation: choose reliable brands, update firmware, separate network
- **Package theft**: deliver to work/PO box/locker, require signature, doorbell camera, "ship to store" pickup, Amazon Key (in-garage delivery if authorized), package receiving box (Parcel Pending, iBot)

## Fire Safety
- **Smoke detectors**: every floor, outside each bedroom, inside each bedroom (if code). Interconnected = all go off together (critical — sleeping person in basement needs to hear attic fire alarm)
  - Test monthly. Replace batteries when clocks change. Replace entire detector every 10 years
  - Dual-sensor: ionization (fast flaming fires) + photoelectric (smoldering fires). Best: dual sensor or both types
  - Kitchen: keep smoke detector at least 10 ft from cooking appliances (reduce false alarms). Use heat alarm for kitchen (doesn't trigger from cooking steam/smoke)
- **CO detectors**: each floor, near bedrooms (carbon monoxide is odorless, deadly, slightly lighter than air — at breathing height). Combination smoke/CO detectors available
  - CO symptoms: headache, dizziness, nausea, confusion, unconsciousness. If alarm goes off: GET OUT, call 911. Don't ignore even if you feel fine
- **Fire extinguisher**: ABC type (all-purpose). One per floor, easily accessible (under kitchen sink, in garage, near furnace). Check pressure monthly (needle in green zone)
  - PASS method: Pull pin, Aim at base, Squeeze handle, Sweep side to side. Recharge after any use (even partial)
  - Kitchen: near exit (not near stove — if fire blocks extinguisher, you can't reach it)
- **Escape plan**: 2 ways out of every room (door + window). Meeting place outside (tree, mailbox, neighbor's). Practice 2x/year with family. Ensure windows open from inside (some painted shut)
  - Sleep with bedroom door closed (slows fire, reduces smoke). Doors protect: closed vs open door = 900°F vs 100°F in room
  - If trapped: seal door cracks with wet towels, call 911, signal at window. Know how to escape second story (collapsible ladder for upstairs, $30-50)
