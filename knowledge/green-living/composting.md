# Composting Guide

## Overview
Composting transforms organic waste into nutrient-rich soil amendment. It reduces landfill waste, methane emissions, and creates free fertilizer.

## Composting Methods

### Backyard Composting (Hot/Cold)
**Hot composting** (fast, 1-3 months):
- 3×3×3 ft minimum pile
- 25-30:1 C:N ratio
- Turn weekly, maintain 130-150°F
- Kills pathogens/weed seeds

**Cold composting** (slow, 6-24 months):
- Add as you go
- Minimal turning
- Lower temps, may not kill seeds

### Vermicomposting (Worm Bins)
- **Red wigglers** (Eisenia fetida)
- Indoor/outdoor, year-round
- Process: 1/2 lb food/day per lb worms
- Harvest: Every 3-6 months
- Output: Worm castings + leachate ("worm tea")

### Bokashi (Anaerobic Fermentation)
- **All food waste**: Meat, dairy, citrus, cooked food
- **Bokashi bran**: EM-1 microbes + bran
- **Process**: Layer food + bran in airtight bucket
- **Output**: Pre-compost (bury in soil 2 weeks) + bokashi tea

### Municipal/Community
- **Curbside pickup**: Check local program
- **Drop-off sites**: Farmers markets, transfer stations
- **Community gardens**: Often accept compost
- **ShareWaste/Olio**: Connect with neighbors

## What to Compost

### Greens (Nitrogen-rich)
- Fruit/vegetable scraps
- Coffee grounds/filters
- Tea bags (remove staples)
- Fresh grass clippings
- Plant trimmings
- Weeds (no seeds)
- Manure (herbivore only)

### Browns (Carbon-rich)
- Dry leaves
- Straw/hay
- Shredded newspaper/cardboard
- Sawdust (untreated wood)
- Wood chips
- Paper egg cartons
- Dryer lint (natural fibers)

### Problem Items (Use Caution)
| Item | Hot Compost | Cold Compost | Vermicompost | Bokashi |
|------|-------------|--------------|--------------|---------|
| Meat/fish | ✓ | ✗ | ✗ | ✓ |
| Dairy | ✓ | ✗ | ✗ | ✓ |
| Oils/fats | Small amounts | ✗ | ✗ | ✓ |
| Citrus/onion | ✓ | ✓ | Limit | ✓ |
| Cooked food | ✓ | ✓ | Small amounts | ✓ |
| Pet waste | ✗ | ✗ | ✗ | ✗ |
| Diseased plants | ✓ (hot only) | ✗ | ✗ | ✓ |
| Weeds with seeds | ✓ (hot only) | ✗ | ✗ | ✓ |
| Glossy paper | ✗ | ✗ | ✗ | ✗ |

### Never Compost
- Meat, fish, bones (attract pests)
- Dairy products
- Oils, fats, grease
- Pet/human waste
- Diseased plants
- Weed seeds
- Treated wood sawdust
- Coal/charcoal ash
- Glossy/coated paper

## Building a Compost Pile

### Layer Method (Lasagna)
```
Bottom: Coarse browns (twigs, straw) - 6" for aeration
Layer 1: Browns - 4-6"
Layer 2: Greens - 2-3"
Layer 3: Soil/compost starter - 1" (inoculates)
Repeat layers
Top: Browns - 4-6" (insulation, odor control)
```

## Maintaining Your Compost

### Moisture
- **Feel test**: Like wrung-out sponge
- **Too dry**: Add water + greens
- **Too wet**: Add browns, turn, improve drainage

### Aeration
- **Turn frequency**: Weekly (hot), monthly (cold)
- **Tools**: Pitchfork, compost aerator, tumbler
- **Signs of anaerobic**: Smell (rotten eggs), slimy, compacted

### Temperature Monitoring
- **Hot pile**: 130-150°F ideal
- **Turn when**: Drops below 100°F or exceeds 160°F
- **Thermometer**: 20" compost thermometer

### Troubleshooting
| Problem | Cause | Solution |
|---------|-------|----------|
| Smells rotten | Too wet, anaerobic | Turn, add browns, improve drainage |
| Smells ammonia | Too much nitrogen | Add browns, turn |
| Not heating | Too small, dry, low N | Build larger, water, add greens |
| Pests (rats, flies) | Meat/dairy, exposed food | Bury food, use closed bin, no meat |
| Too slow | Cold, dry, wrong ratio | Insulate, water, adjust C:N |

## Using Finished Compost

### Readiness Signs
- Dark brown/black, crumbly
- Earthy smell (not sour/ammonia)
- Original materials unrecognizable
- Volume reduced 50%
- Temperature = ambient

### Curing
- Let sit 2-4 weeks after "done"
- Allows microbes to finish
- Reduces phytotoxicity

### Application Rates
| Use | Rate |
|-----|------|
| Garden beds | 1-3" worked into top 6-8" |
| Top dressing | 1/4-1/2" around plants |
| Potting mix | 20-30% compost |
| Lawn topdressing | 1/4" raked in |
| Compost tea | 1:10 dilution, aerated 24hr |
| Seed starting | 10-20% compost (sterilized) |

## Compost Tea
```
Aerated Compost Tea (ACT):
1. 5 gal dechlorinated water
2. 1-2 cups mature compost (in mesh bag)
3. 1-2 tbsp molasses (feeds microbes)
4. Aerate 24-36 hrs (aquarium pump + air stones)
5. Use within 4 hrs

Apply: Soil drench or foliar spray (dilute 1:10)
Frequency: Every 2-4 weeks growing season
```

## Troubleshooting Quick Reference

```
Won't heat up → More greens, water, size, turn
Smells bad → Turn, add browns, check moisture
Flies/gnats → Cover food with browns, bury
Ants → Too dry, water + turn
Mold (white) → Normal, turn in
Mold (black/green) → Too wet, turn + browns
Weeds growing → Not hot enough, screen finished
Still recognizable → Not done, keep composting
```

## Seasonal Composting

### Spring
- Turn pile, add spring greens
- Screen finished compost for beds
- Start new pile

### Summer
- Monitor moisture (dry heat)
- Add browns with grass clippings
- Harvest finished compost

### Fall
- **Best time to start**: Abundant leaves (browns)
- Shred leaves for faster breakdown
- Insulate pile for winter

### Winter
- Insulate: Straw bales, tarp, snow cover
- Keep adding (freeze/thaw helps break down)
- Minimal turning
- Save browns for spring

## Compost Calculators

### C:N Ratio Calculator
```
Target: 25-30:1

Common materials:
- Food scraps: 15:1
- Coffee grounds: 20:1
- Grass clippings: 15-20:1
- Leaves: 40-80:1
- Straw: 80:1
- Sawdust: 200-500:1
- Newspaper: 170:1
- Cardboard: 350:1

Mix calculation:
(C1×W1 + C2×W2) / (W1+W2) = target C:N
```

### Volume Estimator
- **1 cubic yard** = 27 cubic feet = ~200 gallons
- **Typical household**: 500-1000 lbs food waste/year
- **Finished compost yield**: ~30% of input volume
