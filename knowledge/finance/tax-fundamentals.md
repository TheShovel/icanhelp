# Tax Fundamentals (US Federal)

## Tax Brackets 2024 (Single Filers)
```
10%: $0 - $11,600
12%: $11,601 - $47,150
22%: $47,151 - $100,525
24%: $100,526 - $191,950
32%: $191,951 - $243,725
35%: $243,726 - $609,350
37%: $609,351+
```

## Standard Deduction 2024
```
Single: $14,600
Married Filing Jointly: $29,200
Head of Household: $21,900
Additional (65+ or blind): +$1,950 (single) / +$1,550 (married)
```

## Marginal vs Effective Rate
```
Marginal: Rate on LAST dollar earned
Effective: Total tax / Total income
Example: $80k income → 22% bracket, but effective ~13%
```

## Capital Gains Rates 2024
```
Short-term (≤1 year): Ordinary income rates
Long-term (>1 year):
  0%:  Up to $47,025 (single) / $94,050 (MFJ)
  15%: $47,026 - $518,900 / $94,051 - $583,750
  20%: Above those thresholds
+ 3.8% NIIT if MAGI > $200k/$250k
```

## Qualified Dividends
```
Taxed at long-term capital gains rates
Must hold stock >60 days during 121-day period around ex-dividend date
```

## Retirement Contributions 2024
```
401(k)/403(b)/457: $23,000 (+ $7,500 catch-up 50+)
IRA (Traditional/Roth): $7,000 (+ $1,000 catch-up 50+)
HSA: $4,150 individual / $8,300 family (+ $1,000 55+)
SEP-IRA: 25% compensation up to $69,000
Solo 401(k): $69,000 total (employer + employee)
SIMPLE IRA: $16,000 (+ $3,500 50+)
```

## Roth IRA Income Limits 2024
```
Single: Phase-out $146,000 - $161,000
MFJ: Phase-out $230,000 - $240,000
Backdoor: Non-deductible Traditional → Convert (no income limit)
```

## Traditional IRA Deductibility 2024 (Covered by Workplace Plan)
```
Single: Phase-out $77,000 - $87,000
MFJ (spouse covered): $123,000 - $143,000
MFJ (you covered): $230,000 - $240,000
```

## Tax-Advantaged Account Priority
```
1. 401(k) match (100% return)
2. HSA (triple tax advantage)
3. Roth IRA (tax-free growth)
4. 401(k) max (tax-deferred)
5. Taxable brokerage
```

## Common Deductions (Itemized)
```
Medical: >7.5% AGI
SALT: State/local tax up to $10,000
Mortgage interest: Up to $750k debt ($1M pre-2018)
Charity: Cash up to 60% AGI, appreciated stock 30% AGI
Investment interest: Up to net investment income
```

## Above-the-Line Deductions (Reduce AGI)
```
HSA contributions
Traditional IRA (if eligible)
Student loan interest: $2,500 max (phase-out $75k-$90k single)
Self-employed: SE tax deduction, SEP/SIMPLE, health insurance
Educator expenses: $300
Alimony paid (pre-2019 divorce)
Moving expenses (military only)
```

## Tax Credits (Direct Reduction)
```
Child Tax Credit: $2,000/child <17 (phase-out $200k/$400k)
Earned Income Credit: Up to $7,830 (3+ children)
Child/Dependent Care: 20-35% of $3k/$6k expenses
Education: AOTC $2,500 (4 years), LLC $2,000 (unlimited)
Saver's Credit: 10-50% of $2k/$4k contributions (low income)
EV Credit: Up to $7,500 (income/price limits)
Residential Clean Energy: 30% solar/battery
```

## Self-Employment Tax
```
15.3% on net earnings (12.4% SS + 2.9% Medicare)
SS wage base: $168,600 (2024)
Additional 0.9% Medicare >$200k/$250k
Deduct 50% as above-the-line
Quarterly estimated payments required if >$1,000 owed
```

## Business Deductions (Schedule C)
```
Ordinary & necessary expenses
Home office: Simplified $5/sq ft (max 300 sq ft) or actual expenses
Vehicle: Standard mileage $0.67/mi (2024) or actual expenses
Meals: 50% deductible (business purpose)
Travel: 100% transportation/lodging, 50% meals
Equipment: Section 179 (full expensing up to $1.22M), Bonus depreciation 60%
QBI Deduction: 20% qualified business income (phase-out $191k/$383k)
```

## Estimated Tax Payments
```
Due: Apr 15, Jun 15, Sep 15, Jan 15 (next year)
Safe harbor: Pay 100% prior year tax (110% if AGI >$150k)
Or 90% current year tax
Underpayment penalty if short
```

## Education Tax Benefits
```
529 Plan: Tax-free growth for qualified expenses ($10k/yr K-12, unlimited college)
Coverdell ESA: $2k/yr, K-12 + college, income limits
AOTC: $2,500 per student (40% refundable), 4 years
Lifetime Learning: $2,000 per return, unlimited years
Student Loan Interest: $2,500 deduction (phase-out $75k-$90k)
Employer Assistance: $5,250 tax-free
```

## Health Care
```
ACA Marketplace: Premium tax credit (income 100-400% FPL)
HSA: HDHP required ($1,600/$3,200 deductible 2024)
FSA: Use-it-or-lose-it (carryover $640 or grace period)
Medical deduction: >7.5% AGI
LTC Insurance: Age-based deductible limits
```

## Real Estate
```
Primary residence exclusion: $250k/$500k gain (2 of 5 years)
Rental: Depreciate 27.5 years residential, 39 commercial
1031 Exchange: Like-kind deferral (investment property only)
REIT Dividends: 20% QBI deduction eligible
```

## Investment Taxes
```
Wash Sale: Can't claim loss if buy same/substantially identical 30 days before/after
Tax-Loss Harvesting: Offset gains + $3k ordinary income/year
Mutual Fund Distributions: Taxable even if reinvested
ETF: More tax-efficient (in-kind creation/redemption)
Municipal Bonds: Federal tax-free, state tax-free if in-state
```

## Estate & Gift 2024
```
Annual exclusion: $18,000 per recipient
Lifetime exemption: $13.61M (portable to spouse)
Gift tax return: Form 709 if >annual exclusion
Step-up basis: Assets get FMV at death
```

## State Considerations
```
No income tax: AK, FL, NV, SD, TX, WA, WY
NH: Taxes dividends/interest only
Flat tax: CO, IL, IN, KY, MA, MI, NC, PA, UT
High tax: CA (13.3%), NY (10.9%), NJ (10.75%), HI (11%)
Source income: Where work performed
Residency: Domicile + 183 days
```

## Filing Status
```
Single
Married Filing Jointly (usually best)
Married Filing Separately (rarely beneficial)
Head of Household: Unmarried, pay >50% home cost, qualifying dependent
Qualifying Widow(er): 2 years after spouse death, dependent child
```

## Key Deadlines
```
Jan 31: W-2/1099 issuance
Apr 15: Federal return + payment (or extension)
Oct 15: Extended return deadline
Estimated payments: 4/15, 6/15, 9/15, 1/15
```

## Records to Keep
```
3 years: General (statute of limitations)
6 years: Substantial understatement (>25% income)
7 years: Worthless securities, bad debt
Indefinite: Fraud, no return filed
Keep: Tax returns, W-2s, 1099s, receipts, mileage logs, brokerage statements
```

## Tax Planning Strategies
```
Bunch deductions (alternate years)
Harvest losses (offset gains + $3k)
Max pre-tax contributions
Roth conversion in low-income years
Donate appreciated stock (avoid CG + deduction)
Qualified Charitable Distribution (70.5+, up to $100k from IRA)
529 superfunding (5 years × $18k = $90k)
Defer income / accelerate deductions
```

## Audit Triggers
```
High income (>$500k)
Large deductions relative to income
Schedule C losses (hobby loss rule)
Round numbers
Missing 1099s
Home office deduction
Large charitable contributions
Foreign accounts (FBAR >$10k)
```

## Penalties
```
Late file: 5%/month (max 25%)
Late pay: 0.5%/month (max 25%)
Accuracy: 20% underpayment
Fraud: 75%
Estimated tax: ~3% annualized
```

```
MilTax: Military

```
