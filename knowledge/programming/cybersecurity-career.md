# Computer Networks & Cybersecurity Career

## Networking Certifications
- **CompTIA Network+**: entry-level, broad networking fundamentals. For general IT, helpdesk → sysadmin. Vendor-neutral. Covers: TCP/IP, OSI, subnetting, routing protocols (OSPF, EIGRP), switching, wireless, network security, troubleshooting. Good starting point for non-Cisco gear (general IT). Renew: every 3 years by CEUs or retake
- **Cisco CCNA**: more in-depth Cisco-specific (CLI [command-line interface], routing/switching). Entry-level Cisco cert. CCNA: ~1,200-1,500 hours study. Topics: routing, switching, OSPF (EIGRP now removed), STP/RSTP, VLANs, EtherChannel, ACLs, NAT, network automation (Ansible, REST API, Cisco DNA Center, Python). Covers basics of wireless, security, and automation
  - CCNP Enterprise: advanced (300-400 hours). OSPF, BGP, MPLS, VPN, SD-WAN, automation. CCIE: expert-level lab exam (8-hour practical lab), most prestigious in networking. Costs ~$1,600-1,900 for lab attempt. Pass rate: ~40% first attempt. Requires deep real-world experience (not just studying)
  - Renew: CCNA/CCNP every 3 years by CE credits of higher pass
- **Juniper JNCIA**: similar to CCNA but Juniper-specific (JunOS CLI, different philosophy — commit/rollback, not "write mem"). Less common but used in large ISPs, data centers, service providers. JNCIS/JNCIP available for service provider. For Juniper environments: JNCIA, JNCIS
- **Wireshark**: learn packet analysis — how to filter/ decode TCP, HTTP, DNS, etc. Free tool, invaluable for troubleshooting. Display filters: tcp.port == 80, ip.addr == 192.168.1.5 (also and/or/not/comparisons). Color coding rules. Follow TCP stream. Export objects. tshark command line version
- **Linux networking**: iproute2 tools (ip addr, ip route, ip neigh, ip link). SS replaces netstat (ss -tulpn). iptables/nftables basics. DNS: /etc/resolv.conf, systemd-resolved, nsswitch.conf. NetworkManager: nmcli. traceroute/ping/curl/mtr tools. netfilter: iptables chain ordering (rules in order, match first then jump to target) — filter (INPUT, FORWARD, OUTPUT), nat, mangle. iptables -A INPUT -p tcp --dport 22 -j ACCEPT

## Cybersecurity Career Path

### Entry Level
- **CompTIA Security+**: baseline security certification for DoD 8570 compliance (see IAT/IAM/IASAE requirements). Risk management, cryptography, identity & access management (IAM), secure architecture, threats/vulnerabilities, incident response, governance/privacy. Vendor-neutral. Good starting security cert. Renew every 3 years
- **Blue team**: SOC analyst (defensive — monitor alerts, triage incidents, malware analysis). Tools: SIEM (Splunk, QRadar, Elastic Security — Splunk maybe most common, Blue Team must learn SPL searching, correlation rules, alert tuning, incident handling). EDR (CrowdStrike, SentinelOne, Microsoft Defender for Endpoint). Phishing analysis. Digital forensics: memory analysis (Volatility, Rekall), disk forensics (Autopsy, FTK Imager, dd for Linux, strings for scanning entire drive for artifacts)
  - SOC tiers: T1 (triaging alerts — low criticality, tuning false positives). T2 (investigation of real incidents). T3 (hunting/threat intel/ advanced forensics/malware reverse engineer). Some SOCs have dedicated threat intel team
- **Red team**: penetration tester — ethical hacking. Cert: OSCP (Offensive Security Certified Professional — 24-hour practical exam, 70 points out of 100 to pass. Hardest entry-level cert. Requires: exploit buffer overflows, web app attacks, Active Directory attacks, privilege escalation, report writing). Also: GPEN (GIAC Penetration Tester), PNPT (TCM Security's Practical Network Penetration Tester — more modern, includes AD pentesting)
  - Skills: Kali Linux, Metasploit, Burp Suite (web app scanning), Nmap, Nessus, GoBuster for directory enumeration, hydra for brute forcing, sqlmap for SQL injection, BeEF for XSS, BloodHound for AD, Responder for LLMNR/NBT-NS poisoning, Cobalt Strike/ActiveBreach for post-exploitation/ C2
  - Active Directory is huge topic: almost 90% of Fortune 500 companies use AD for authentication. Red team: exploit Kerberos/Kerberoasting, AS-REP roasting, DCSync, NTLM relay, ACL abuse, GPO abuse. BloodHound: graph Active Directory relationships to find privilege escalation paths. Mimikatz: extract plaintext passwords/ hashes/kerberos tickets from memory (must be admin). CrackMapExec (CIMPLEX): powershell remoting + lateral movement + credential reuse tests across network
  - Phishing: social engineering for initial access. GoPhish, SET, Evilginx (reverse proxy for MFA bypass). Phishing engagement: send phish → collect user click/password → report on awareness weakness. MFA can be bypassed: proxy + relay + session cookie replay via Evilginx. For testing better auth: MFA fatigue push bombing = bomb user until they allow push notification

### Mid Level
- **Blue**: security engineer (implement security controls). Incident responder (lead investigations — MITRE ATT&CK framework mapping). Threat hunter (proactive detection — hypothesis-driven, often using EDR data + threat intel)
  - Cert: CISSP (Certified Information Systems Security Professional) — management cert required for many senior security roles. 10 domains: Security & Risk Mgmt, Asset Security, Security Architecture, Comm & Network Security, IAM, Assessment & Testing, Security Operations, Software Development Security. 5 years paid experience required, experience requirement measured across 2+ domains. "Associate of ISC2" if not yet 5 years experience
- **Red**: senior pentester (specialize in web app, mobile, network, cloud, hardware, social engineering). Red team operator (simulate full adversary — extended engagement, not just technical find-and-report). Cert: OSCE (300, OSEE for exploit dev). SANS GPEN, GXPN (exploit researcher), GWAPT (web app pentest). Also Certified Red Team Operator (CRTO) from Pentester Academy for C2 frameworks

### Advanced
- **Security architect**: design secure systems. Cloud security architect. Cert: CISSP-ISSAP (architecture), CCSK/CCSP (cloud security). SANS GIAC. 10+ years experience. Understand cryptography, federation, identity management, zero trust, network segmentation, threat modeling, cloud design patterns (CSPs: Azure, AWS, GCP)
- **CISO**: Chief Information Security Officer. Executive role — manage security program, budget, compliance, board communication, vendor management, incident response leadership. Blend of technical + business + management. CISSP + CISM (Certified Information Security Manager, ISACA — mgmt focused). 10+ years in various security roles. Many CISOs in Fortune 500 have MBA, excellent communication + risk management skills

## Key Security Skills

### Cloud Security
- **Shared responsibility model**: AWS/Azure/GCP secures the cloud (physical, hypervisor, network). YOU secure what's IN the cloud (OS, app, data, access). Customer is always responsible for data classification, access controls, compliance. Provider always responsible for physical infrastructure
  - For SaaS: mostly the cloud provider's job (but customer responsible for user access, data classification). For IaaS: heavily the customer's job (patching VMs, configuring firewalls, IAM, encryption)
- **IAM (Identity & Access Management)**: least privilege — minimal permissions possible, for shortest time. AWS IAM roles, Azure AD. Policy as code (Terraform, CloudFormation). Zero trust: never trust, always verify (no inherent trust from network location). Conditional access policies (geo, device, risk)
  - Common misconfigurations: overly permissive roles, long-term access keys, unused roles, public S3 buckets. Biggest cloud security failures: misconfigured IAM + public S3 buckets + leaked keys in code (GitHub)
- **CASB (Cloud Access Security Broker)**: between users and cloud apps — visibility, compliance, data security. Microsoft Defender for Cloud Apps, Netskope, Zscaler. Find Shadow IT (unauthorized cloud services used by employees). DLP (Data Loss Prevention) to scan + block data leaving cloud apps
- **CWPP (Cloud Workload Protection)**: agent on VMs/containers to scan + protect against threats. Runtime protection, vulnerability scanning (Qualys, Rapid7, Prisma Cloud, Wiz/CrowdStrike)
- **Kubernetes security**: RBAC (limit who can access API server — avoid overly permissive cluster-admin). Pod Security Standards (privileged/compliant/restricted). Network policies (limit pod-to-pod communication). Secrets encryption. Container image scanning (Trivy, Anchore, Snyk). Admission controllers (Kyverno, OPA/Gatekeeper): enforce policies (block privileged containers, enforce read-only root filesystem, require resource limits)

### Web Application Security
- **OWASP Top 10**: most critical web app risks. Current (2021): Broken Access Control, Cryptographic Failures, Injection, Insecure Design, Security Misconfiguration, Vulnerable Components (known vuln — supply chain), Identification+Auth Failures, Software+Data Integrity, Logging+Monitoring, SSRF
  - #1 Broken Access Control consistently at top — most widespread (insufficient authorization checks, IDOR: Insecure Direct Object Reference — trusting user-supplied ID to access another user's resource)
- **DAST** (Dynamic Application Security Testing — ZAP, Burp Suite, AppScan): scanning running application, finds live vulnerabilities (like XSS, SQLi). Good finder of runtime issues. SAST (Static — Semgrep, SonarQube, Checkmarx, Fortify): scanning source code. Finds vulnerabilities in code (hardcoded secrets, dangerous API (sink/taint) misuse). DAST + SAST cover different things. Use both
- **SCA** (Software Composition Analysis — Snyk, Dependabot, OWASP DC): scan open-source libraries for known vulnerabilities. Supply chain risk: SolarWinds breach (2020, state-sponsored via compromised build system). Log4j (2021, critical RCE in logging library, every Java app at risk for weeks. CVE-2021-44228: one line of input triggers JNDI lookup → attacker code execution. Ultimate nightmare: everywhere, easy to exploit, hard to patch all copies individually)
- **Bug bounty**: HackerOne, Bugcrowd — paid ethical hacking. Companies post bounties for finding vulnerabilities. Ranges: $500 SQLi → $100,000+ RCE. Learn from reports (read public disclosed reports). Good way to gain real experience and portfolio

### Incident Response (IR)
- **Phases**: Preparation (IR plan, tools, team, playbooks. Tabletop exercises). Detection (SIEM alerts, EDR detection, user reports, threat intel feeds). Analysis (triage: false positive? Confirm true positive. Containment scope: isolate affected systems). Containment (disconnect from network, isolate VM, disable account, sinkhole domain). Eradication (remove malware, patch vulnerability, rebuild affected systems). Recovery (bring back online — restore from clean backup, rotate all credentials — avoid re-infection). Lessons learned (post-mortem — what went well? What should change? Update playbooks, detection rules, training)
- **MITRE ATT&CK**: framework of adversary tactics + techniques. Blue team use: map detections + response to ATT&CK. Red team use: model adversary behavior. Purple team: working together to improve detections
- **DFIR** (Digital Forensics + Incident Response): image disk (FTK Imager, dd — capture disk as raw image file). Memory capture (RAM — volatility analysis: processes, network connections, injected code). Timeline analysis (super timeline — when did files appear, get modified, registry changes). Malware analysis: static (strings, hash), dynamic (run in sandbox, record behavior). Windows artifacts: NTFS journal (USN journal — logs all file changes), $MFT (Master File Table), event logs (security.evtx, system.evtx), registry (NTUSER.DAT, SAM, SYSTEM, SOFTWARE), prefetch files (track app execution), LNK files (shortcuts to recently accessed files), browser history

## Certifications Path Summary
- Entry: Sec+ → CySA+ (CompTIA) or SSCP (ISC2)
- Technical: CCNA → Security + Experience → CISSP (or try SSCP earlier) → CCNP Security
- Red team: eJPT (elearnSecurity junior) → OSCP (crucial) → PNPT → OSCP → OSCE/OSED/OSEP or CRTP/CRTE (Active Directory pentesting, Pentester Academy)
- Cloud: AWS CCP → AWS SA Associate → AWS Security Specialty → CCSK/CCSP
- Management: CISSP (+ CISM) → CISA (audit) → CRISC (risk). Also: MBA optional
