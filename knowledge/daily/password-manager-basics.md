# Password Manager Basics

## Why Use One
A password manager lets every site have a unique strong password. This prevents one breached website from exposing every other account. Memorize one long master password and store the rest in the vault.

## Master Password Rules
- Use a long passphrase, ideally 4–6 random words or 16+ random characters.
- Do not reuse your email, device, banking, or work password.
- Write the master password on paper and store it somewhere safe until you are certain you remember it.
- If supported, enable account recovery carefully; weak recovery can weaken the vault.

## Two-Factor Authentication
- Use authenticator apps or hardware security keys when possible.
- SMS is better than no 2FA, but weaker than app-based codes or passkeys.
- Save backup codes in the password manager and also keep an offline copy for critical accounts.
- Protect email first: whoever controls your email can reset many other passwords.

## Migration Checklist
1. Install a reputable password manager on phone and computer.
2. Create a strong master password.
3. Add email accounts first and change their passwords.
4. Add banking, cloud storage, social media, shopping, and work accounts.
5. Replace reused passwords gradually, starting with important accounts.
6. Turn on 2FA for important accounts.
7. Delete plain-text password notes and browser-saved duplicates after confirming imports.

## Creating Good Passwords
Use the generator built into the manager. Prefer 16–24+ random characters for websites. Use longer passphrases for passwords you must type manually on TVs, consoles, or shared devices.

## Sharing Passwords Safely
Use the manager’s sharing feature instead of texting or emailing passwords. Shared vaults let access be revoked and rotated. For one-time sharing, use an expiring secret link only if the service supports it securely.

## Emergency Access
For family or business accounts, document emergency access. Include the password manager name, where recovery codes are stored, and how to access important devices. Do not put the master password in an unprotected cloud note.

## Warning Signs
- Browser popup asks for the master password on a strange page.
- A site asks for your 2FA recovery code during normal login.
- Password manager extension is missing or has the wrong publisher.
- You receive password reset emails you did not request.
If this happens, stop, verify the URL, check account sessions, and change passwords from a trusted device.
