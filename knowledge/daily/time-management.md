# Time Management

Methods and frameworks for prioritization, scheduling, and productivity.

## Eisenhower Matrix
- **Quadrant 1: Urgent & Important** — Do first: crises, deadlines, pressing problems
- **Quadrant 2: Not Urgent & Important** — Schedule: planning, exercise, relationships, skill development
- **Quadrant 3: Urgent & Not Important** — Delegate: interruptions, some emails, meetings
- **Quadrant 4: Not Urgent & Not Important** — Eliminate: social media scrolling, busywork, TV
- Spend most time in Q2 — proactive work prevents Q1 crises

## Getting Things Done (GTD)
- **Capture**: Write everything (tasks, ideas, commitments) into trusted inbox — clear your mind
- **Clarify**: Process each item — Is it actionable? If no → trash, incubate (someday/maybe), or reference. If yes → what's next action?
- **Organize**: Projects (multi-step outcomes) → Next Actions (single steps) → Waiting For (delegated) → Calendar (time-specific) → Someday/Maybe
- **Reflect**: Weekly review — empty inboxes, review projects, update lists, clean calendar
- **Engage**: Work from contexts (@home, @computer, @errands) — choose by energy, time, priority
- Key rule: If a task takes <2 minutes, do it immediately

## Time Blocking
- **Schedule tasks** instead of listing to-dos — assign specific time slots on calendar
- Task blocks: 60-120 min for deep work, 15-30 min for shallow work (email, admin)
- **Theme days**: Monday=Planning, Tuesday=Deep Work, Wednesday=Meetings, Thursday=Coding, Friday=Review
- **Buffer time**: Leave 25-50% of day unscheduled for interruptions and overflow
- **Day theming**: Elon Musk blocks in 5-minute increments; Bill Gates in 5-minute; Cal Newport in 60-90 minute deep blocks

## Pickle Jar Theory
- **Rocks** (big important tasks): Strategic projects, relationships, health — schedule first
- **Pebbles** (medium tasks): Meetings, reports, emails — fit around rocks
- **Sand** (small tasks): Fillers — idle time, commute, waiting — fill remaining gaps
- If you fill the jar with sand first (small busywork), there's no room for rocks (important work)

## Pomodoro (Detailed)
- **25 min focus** → 5 min break — after 4 pomodoros, take 15-30 min break
- Each pomodoro is indivisible — if interrupted, restart the pomodoro
- Track interruptions on a sheet — review later, reduce sources
- Adjust timing: Some use 52 min work / 17 min break (ultradian rhythm)
- Apps: TomatoTimer, Focus Booster, Toggl, Forest

## Eat the Frog
- **Do the hardest, most important task first** — before checking email, before distractions
- Willpower decreases throughout the day — front-load cognitively demanding work
- "If you have to eat two frogs, eat the ugliest one first" — Mark Twain

## 2-Minute Rule
- If a task takes ≤2 minutes, do it now — reply to email, wash dish, file document
- Prevents tiny tasks from accumulating into overwhelming backlog
- For bigger tasks: "Start for 2 minutes" — often leads to continuing past 2 minutes

## Parkinson's Law
- **Work expands to fill the time available** — set tighter deadlines for efficiency
- If you have 8 hours for a task that needs 2, you'll use all 8 hours
- Solution: Set artificial deadlines — "I'll finish this report by 10 AM" not "by end of day"
- Use timers for tasks to stay focused and on pace

## Kanban for Personal Productivity
- **Columns**: To Do → Doing → Done — visual workflow on board (physical or digital)
- **WIP limit**: Maximum items in "Doing" column (usually 1-3) — prevents multitasking
- Pull new work only when current task is complete
- Tools: Trello, Notion, GitHub Projects, physical Post-It notes on whiteboard

## Energy Management
- **Match tasks to energy levels**: High-energy windows → deep work; low-energy → admin, email
- **Chronotypes**: Lions (morning peak), Bears (midday peak), Wolves (evening peak), Dolphins (erratic)
- **Ultradian rhythm**: 90-120 min focus cycles — brain needs break after each cycle
- **Biobreaks**: Every 90 min — walk, stretch, hydrate, look at distance (prevents eye strain)
