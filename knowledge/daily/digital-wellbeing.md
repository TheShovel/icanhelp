# Digital Wellbeing

## Screen-Time Boundaries
Attention is the scarce resource; treat it like a budget. Default-open devices erode focus without visible cost.

```
set app limits        — iOS Screen Time / Android Digital Wellbeing: hard daily caps
grayscale mode        — desaturate screen; dopamine pull of color drops sharply
phone-free bedroom    — charge outside; buy a $15 alarm clock instead
one-screen rule       — no second device while watching TV or eating
weekly review         — check weekly report; notice creep before it compounds
```

Aim for a "deadline" hour: screens off 60–90 min before sleep to protect rest.

## Notification Hygiene
Most pings are someone else's priority, not yours. Each interrupt costs ~23 min to refocus (per UC Irvine research).

```
batch notifications    — deliver in 2–3 scheduled bursts, not instantly
disable badges         — red counts trigger compulsive checking; turn off
allow only calls + 3 apps — everything else silent; review weekly
use Do Not Disturb     — auto-schedule during focus, sleep, meals
mute group chats       — leave or silence; catch up on your schedule
```

Audit: if a notification hasn't changed your action in a week, it doesn't deserve to buzz.

## Focus Blocks & Deep Work
Protect contiguous time; context-switching is the real productivity tax.

```
time-block calendar    — 90-min deep blocks, 2–3 per day max
website blocker        — Freedom / Cold Turkey during blocks
single-tab rule        — one task, one tab; close the rest
Pomodoro 52/17         — 52 min focus, 17 rest (or 25/5 for lighter tasks)
status signal          — shared "heads-down" status with household/coworkers
```

Deep work shrinks when fragmented; guard the first 2 hours of your day for the hardest task.

## Blue Light & Sleep
Evening screen light suppresses melatonin, delaying sleep onset by ~30–60 min.

```
night shift / f.lux    — warm screen after sunset (not a full fix, but helps)
no screens 60 min pre-bed — read, stretch, or talk instead
blue-blocker glasses   — amber lenses in evening reduce melatonin suppression
dim ambient lights      — bright rooms signal "day" to circadian clock
```

Sleep debt is cumulative; one late-night scroll can cost two days of recovery.

## Digital Declutter Cadence
Regular pruning keeps the system calm. Match cadence to the mess.

```
daily  — inbox zero or close; clear 1 notification tray
weekly — uninstall unused apps; unsubscribe from 5 emails
monthly— review permissions; delete old photos/screenshots
quarterly— audit subscriptions; export + delete dead accounts
yearly — full device reset/refresh; re-evaluate tools you pay for
```

Unsubscribe aggressively; a lean inbox is a low-stress inbox.

## Attention & Usage Tracking
You can't manage what you don't measure.

```
built-in trackers      — Screen Time / Digital Wellbeing show category breakdowns
RescueTime-style       — automatic categorization of productive vs distracting
manual log             — note "why" you opened the phone; reveals triggers
set intentions         — name the task before unlocking; reduces mindless use
```

Aim to discover one app that eats >1 hr/day you didn't intend. That's the first fix.

## Email Batching
Continuous email is a focus killer dressed as work.

```
2–3 send times         — process inbox at fixed slots, not continuously
Inbox Zero method      — delete / delegate / do (<2 min) / defer
templates              — canned replies for repeated asks
turn off email push     — fetch manually during batch windows
use rules/filters       — auto-sort newsletters, receipts, alerts
```

Batching cuts the "check email" reflex that fragments every other task.

## Common Problems → Fix
| Problem | Likely Cause | Fix |
|---------|--------------|-----|
| Can't sleep | late screen + blue light | grayscale + 60-min cutoff |
| Constant distraction | instant notifications | batch + DND |
| Lost evenings to scroll | infinite feed | app limit + grayscale |
| Email overwhelm | continuous checking | 2–3 batch slots |
| Eye strain | no breaks | 20-20-20: look 20ft, 20s, every 20 min |
| Low focus | fragmented mornings | protect first 2 hours |
