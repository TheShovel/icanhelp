# Memory Techniques

Methods for improving encoding, storage, and retrieval of information.

## Mnemonic Systems
- **Acronyms**: Create words from first letters — ROYGBIV (colors of rainbow: Red, Orange, Yellow, Green, Blue, Indigo, Violet)
- **Acrostics**: Sentences where first letters spell target — "My Very Educated Mother Just Served Us Noodles" (planets order)
- **Rhymes & songs**: "In fourteen hundred ninety-two, Columbus sailed the ocean blue" — rhythm makes sticky
- **Keyword method**: Link foreign word to similar-sounding English word + image — French "pomme" (apple) → "Pom" the dog eating an apple
- **Chunking**: Group digits into meaningful clusters — credit card 1234-5678-9012-3456 easier than 12 digits alone

## Method of Loci (Memory Palace)
- **Choose a familiar place**: Your home, daily commute route, school hallway — must know it vividly
- **Define a path**: Walk through in consistent order — front door → living room → kitchen → bedroom
- **Place each item at a locus**: Associate concept with specific location — imagine milk spraying from chandelier
- **Make images bizarre, vivid, emotional**: Exaggerated, moving, silly images are more memorable
- **Walk through mentally**: Retrace the path to recall each item in order
- Use for: speeches (walk through points), grocery lists, exam topics, presentations

## Peg System
- **Peg words**: Pre-memorize "one = bun, two = shoe, three = tree, four = door, five = hive, six = sticks, seven = heaven, eight = gate, nine = line, ten = hen"
- **Link each item**: Associate item 1 with bun (interactive image), item 2 with shoe, etc.
- **Recall**: Run through pegs 1-10 to retrieve items in order
- Advantage over Loci: Works for numbered lists (top 10, chronological events)

## Spaced Repetition
- **Forgetting curve** (Ebbinghaus): Memory decays exponentially — recall drops ~50% in 1 hour, ~30% in 1 day
- **Optimal intervals**: Reviews just before forgetting — strengthens long-term retention
- **Leitner system**: Boxes with increasing intervals:
  - Box 1: review daily
  - Box 2: review every 2 days  
  - Box 3: review weekly
  - Box 4: review biweekly  
  - Box 5: review monthly
  Correct answer moves card up; incorrect drops back to Box 1
- **Digital**: Anki, Memrise, SuperMemo — algorithms calculate optimal review time per card

## Encoding Strategies
- **Elaborative rehearsal**: Connect new info to existing knowledge — "How is this like X? How is it different from Y?"
- **Self-explanation**: Explain concepts aloud in your own words — identifies gaps in understanding
- **Dual coding**: Combine verbal (words) with visual (diagrams, charts, mind maps) — dual memory traces
- **Concrete examples**: Abstract concept + specific case study — "Recursion is like Russian nesting dolls"
- **Teaching others**: Feynman technique — explain as if teaching a child, identify gaps, simplify

## Sleep and Memory
- **NREM sleep (deep sleep)**: Consolidates declarative memories (facts, events) — hippocampus replays day's learning
- **REM sleep**: Consolidates procedural memories (skills, habits) and integrates with existing knowledge
- **Naps**: 20-min power nap boosts alertness; 90-min full-cycle nap improves memory consolidation
- **Before bed**: Review material right before sleep — less interference during consolidation
- **Morning recall**: Test yourself after waking — sleep has processed the information

## Interleaving
- **Mix topics** during practice instead of blocking one topic at a time
- Example: Study quadratic equations, then trigonometry, then logarithms in same session
- Slows learning initially but improves long-term retention and transfer
- Forces brain to identify problem types rather than mindlessly applying same formula
