# Essential Knots for Everyday Use

## Overview
A few well-chosen knots handle 95% of daily tying needs. Focus on: easy to tie, easy to untie after load, secure, and memorable.

## Knot Terminology
- **Standing part**: Long end, not actively used
- **Working end**: Short end, used to tie
- **Bight**: U-shaped bend (not crossed)
- **Loop**: Crossing turn
- **Turn**: Around object
- **Round turn**: Full wrap + crossing
- **Standing end**: Fixed end
- **Running end**: Free end

## The Essential 8 Knots

### 1. Square Knot (Reef Knot) - Joining Two Ropes
```
Use: Joining two ropes of equal diameter, first aid, bundling
NOT for: Critical loads, different diameters, slippery rope

Tie: Right over left, left over right (or vice versa)
Mnemonic: "Right over left, left over right"
Check: Both standing parts exit same side
Untie: Push loops toward each other
```

### 2. Sheet Bend - Joining Unequal Ropes
```
Use: Different diameters, rope to tarp grommet, extending lines
Better than: Square knot for unequal ropes

Tie: Bight in thicker rope, pass thinner up through, around back, tuck under itself
Double sheet bend: Extra wrap for security/slippery rope
```

### 3. Bowline - Fixed Loop (King of Knots)
```
Use: Rescue, mooring, tie-down anchor, harness
Never slips, unties easily after heavy load
One-handed tie possible

Tie: "Rabbit comes out of hole, around tree, back down hole"
1. Form small loop (hole) standing part on top
2. Working end up through loop (rabbit out)
3. Around standing part (around tree)
4. Back down through loop (back down hole)
5. Tighten: pull standing part while holding loop
```

### 4. Taut-Line Hitch - Adjustable Loop
```
Use: Tent guy lines, tarp ridges, clotheslines, any adjustable tension
Slides when loose, locks under tension

Tie: Around standing part, inside loop 2-3 turns, outside 1 turn
Direction: Working end exits same direction as standing part
Adjust: Slide hitch along standing part
Lock: Pull working end tight
```

### 5. Trucker's Hitch - Mechanical Advantage (3:1)
```
Use: Securing loads, tarps, kayaks, tight lines
Creates 3:1 mechanical advantage

Tie:
1. Fixed anchor (bowline or round turn + 2 half hitches)
2. Form slip knot loop in standing part (pulley)
3. Pass working end through anchor, back through loop
4. Pull tight (3:1 advantage), secure with 2 half hitches
```

### 6. Clove Hitch - Quick Attachment
```
Use: Temporary attachment to pole/post, starting lashings
Quick, adjustable, but can slip on smooth/square objects

Tie: Wrap around object, cross over, wrap again, tuck under cross
Or: Two loops in hand, slide over post
Secure: Add half hitches (round turn + 2 half hitches = secure)
```

### 7. Round Turn + Two Half Hitches - Secure Attachment
```
Use: Mooring, anchoring, any secure fixed tie
Strong, secure, unties after load

Tie:
1. Round turn (full wrap + cross) around object
2. Half hitch around standing part
3. Second half hitch same direction
3. Snug each hitch before next
```

### 8. Prusik Knot - Friction Hitch (Ascending/Backup)
```
Use: Climbing backup, ascending rope, gripping slippery line
Slides when loose, locks under load

Tie: Loop of cord (Prusik loop), 3-4 wraps around main line
Through itself each wrap
Dress: Wraps neat, parallel
Load: Pull loop - locks; push knot - slides
```

## Specialized but Useful

### Figure-8 Loop - Stronger Fixed Loop
```
Use: Climbing, critical loads, where bowline might shake loose
Stronger than bowline, easy to inspect

Tie: Figure-8 in standing part, retrace with working end
Dress: Parallel strands, no crossings
```

### Alpine Butterfly - Mid-line Loop
```
Use: Isolate damaged section, mid-line anchor, haul systems
Loads in any direction, unties after load

Tie: Form loop, twist twice, wrap around standing parts, tuck
```

### Constrictor Knot - Permanent Binding
```
Use: Bundling, hose clamps, permanent binding
"Never comes untied" - often must be cut

Tie: Like clove hitch but working end tucked under both wraps
```

### Sheet Bend (Double) - Net Making/Tarp Corners
```
Use: Joining multiple lines, net repair, tarp corners
Double version: Extra wrap for security
```

### Rolling Hitch - Gripping Another Rope
```
Use: Pulling along a line, tensioning, adjustable attachment
Directional: Only holds one way

Tie: 2 wraps inside, 1 wrap outside, like taut-line but directional
```

## Rope/Cord Selection

### Materials
| Material | Strength | Stretch | UV Resistance | Best For |
|----------|----------|---------|---------------|----------|
| **Nylon** | High | High (15-25%) | Good | Dynamic loads, anchor lines |
| **Polyester (Dacron)** | High | Low (<10%) | Excellent | Static loads, guy lines, marine |
| **Polypropylene** | Medium | Medium | Poor | Floating, cheap temporary |
| **Dyneema/Spectra** | Highest | Minimal | Good | Ultralight, high strength |
| **Natural (Manila, Sisal)** | Low | Low | Poor | Decorative, gardening |

### Sizing (Diameter vs Working Load)
| Diameter | Nylon WLL | Polyester WLL | Common Use |
|----------|-----------|---------------|------------|
| 3 mm (1/8") | 100 lbs | 120 lbs | Paracord, guy lines |
| 5 mm (3/16") | 300 lbs | 350 lbs | General utility |
| 6 mm (1/4") | 500 lbs | 600 lbs | Tent lines, tie-downs |
| 10 mm (3/8") | 1,500 lbs | 1,800 lbs | Anchor, vehicle recovery |
| 12 mm (1/2") | 2,500 lbs | 3,000 lbs | Heavy loads |

**Safety factor**: 5:1 to 10:1 (WLL = Breaking Strength ÷ Safety Factor)

## Knot Tying Principles

### Dressing & Setting
1. **Dress**: Arrange strands parallel, no crossing
2. **Set**: Apply load gradually to seat knot
3. **Inspect**: Correct structure, tails adequate

### Tail Lengths
- **Security knots**: 6-12 inches
- **Load-bearing**: 12-18 inches
- **Critical**: 18+ inches or stopper knot

### Stopper Knots (Prevent Pull-through)
- **Figure-8**: Standard, easy to inspect
- **Double overhand**: Bulkier, more secure
- **Ashley stopper**: Maximum bulk

## Quick Reference Card (Memorize These Patterns)

```
BOWLINE:     Hole → Rabbit → Tree → Hole
TAUT-LINE:   Inside 3x → Outside 1x → Dress
TRUCKER:     Anchor → Loop → Through → Pull → Hitch
SHEET BEND:  Bight → Up → Around → Under
CLove HITCH: Wrap → Cross → Wrap → Tuck
PRUSIK:      Loop → Wrap 3x → Through → Dress
```

## Safety Notes
- **Never** stand in bight of loaded line
- **Never** wrap rope around body/limbs
- **Inspect** rope before each use (cuts, abrasion, UV damage)
- **Retire** rope after major load, chemical exposure, 5+ years
- **Learn** proper coiling (figure-8, butterfly) to prevent kinks
- **Practice** with same rope/cord you'll use in real situations
