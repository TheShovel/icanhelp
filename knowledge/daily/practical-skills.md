# Daily Skills Reference

## Car Maintenance Basics

### Monthly Checks
- [ ] Tire pressure (cold, per door jamb sticker)
- [ ] Tire tread (penny test: Lincoln's head covered = OK)
- [ ] Oil level (dipstick, on level ground)
- [ ] Coolant level (reservoir, cold engine)
- [ ] Windshield washer fluid
- [ ] Lights: head, brake, turn, reverse
- [ ] Wiper blades (streaking = replace)

### Every 3-6 Months
- [ ] Oil change (per manual: 5k-10k miles or 6-12 months)
- [ ] Rotate tires (5k-7k miles)
- [ ] Brake pad inspection
- [ ] Battery terminals (clean corrosion)
- [ ] Air filter (engine & cabin)
- [ ] Belts & hoses (cracks, wear)

### Fluids & Intervals
```
Engine oil:       5,000-10,000 mi / 12 mo
Transmission:     30,000-60,000 mi (check manual)
Coolant:          30,000-50,000 mi / 5 yr
Brake fluid:      20,000-45,000 mi / 2-3 yr
Power steering:   50,000-100,000 mi
Differential:     30,000-60,000 mi
```

### Warning Lights
```
Red = Stop immediately (oil, brake, battery, temp)
Yellow = Service soon (check engine, ABS, TPMS, oil change)
Green/Blue = Info (high beams, cruise, eco mode)
```

### Common Fixes

#### Leaky Faucet
```
1. Shut off water (under sink or main)
2. Plug drain
3. Remove handle (set screw or cap)
4. Remove cartridge/valve
5. Replace O-rings/washers/cartridge
6. Reassemble, test
```

#### Running Toilet
```
Check: Flapper seal (most common), fill valve, chain length
Fix: Replace flapper ($5), adjust chain, or replace fill valve ($15)
```

#### Clogged Drain
```
1. Plunger (flange for toilet, cup for sink)
2. Drain snake / zip-it tool
3. Baking soda + vinegar + hot water (maintenance)
3. Chemical (last resort, damages pipes)
4. Clean P-trap (bucket underneath)
```

#### Drywall Patch (Small Hole)
```
1. Clean edges, sand
2. Self-adhesive mesh patch
3. Joint compound (2-3 thin coats)
4. Sand between coats
5. Prime, paint
```

#### Outlet/Switch Replacement
```
1. TURN OFF BREAKER (verify with tester)
2. Remove cover plate
3. Unscrew device, pull out
4. Note wire positions (photo)
5. Disconnect, connect new same way
   Black/brown/red → brass (hot)
   White → silver (neutral)
   Green/bare → green (ground)
6. Secure, test, replace cover
```

### Knife Skills
```
Chef's knife: 90% of tasks
Grip: Pinch blade base, curl fingers (claw)
Rocking motion, tip on board
Sharp = safe (less force = more control)
Hone weekly, sharpen 1-2x/year
```

### Temperature Basics
```
Beef/Lamb: Rare 125, Med-rare 135, Med 145, Well 160°F
Pork: 145°F + 3 min rest
Chicken/Turkey: 165°F (breast), 175°F (thigh)
Ground meats: 160°F
Fish: 145°F or opaque/flakes
Leftovers: 165°F
Danger zone: 40-140°F (max 2 hrs, 1 hr if >90°F)
```

### Food Storage
```
Fridge: 40°F or below
Freezer: 0°F or below
Fridge zones: Coldest back bottom, warmest door
Meat: Bottom shelf (prevent drips)
Produce: Crisper drawers (humidity control)
Leftovers: 3-4 days max
Label & date everything
```

## Digital Security

### Password Hygiene
```
Password manager (Bitwarden, 1Password, KeePass)
Unique 16+ char for every site
2FA on: Email, banking, social, password manager
Passphrase method: 4-5 random words (correct-horse-battery-staple)
```

### Device Security
```
Auto-updates ON (OS, apps, browser)
Full disk encryption (BitLocker, FileVault, LUKS)
Screen lock: 5 min timeout, PIN/biometric
Find My Device enabled
Backup: 3-2-1 rule (3 copies, 2 media, 1 offsite)
```

### Network
```
Router: Change admin password, update firmware
WPA3 or WPA2-AES (not TKIP)
Guest network for IoT/visitors
DNS: 1.1.1.1 or 9.9.9.9 (privacy)
VPN on public WiFi
```

### Phishing Red Flags
```
Urgent/threatening language
Generic greetings
Poor grammar/spelling
Mismatched URLs (hover to check)
Requests for credentials/payment
Unexpected attachments
```
