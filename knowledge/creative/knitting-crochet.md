# Knitting & Crochet Basics

## Knitting vs Crochet
- **Knitting**: two needles, loops on needle, elastic fabric, good for sweaters/socks/hats — more draping, uses less yarn per project
  - Knit stitch = V shapes, purl = bumps. Combinations create patterns (stockinette, rib, seed)
  - Better for: garments that need stretch (socks, sweaters, hats), fine gauge, drapey fabrics
- **Crochet**: one hook, one loop on hook at a time, thicker/ stiffer fabric, good for blankets/amigurumi/toys — faster, easier to fix mistakes
  - Stitches vary in height (single, half-double, double, treble)
  - Better for: structured items (baskets, bags, stuffed animals), lace patterns, thick blankets, granny squares

## Knitting Essentials
- **Materials**: yarn (weight matters — lace, fingering, sport, DK, worsted, bulky, super-bulky), needles (straight, circular, double-pointed — sizes US 0-50, mm 2-25)
  - Needle size determines gauge: bigger needles + thicker yarn = looser, faster project
  - Circular needles can be used for flat or round knitting — most versatile, buy 16" and 32-40" lengths
- **Cast on**: put yarn on needle to start — long-tail cast on (most common, stretchy edge), knitted cast on (simple, beginner), cable cast on (firm, decorative)
- **Knit stitch**: right needle through front of left loop, yarn over, pull through, slip off left
- **Purl stitch**: yarn in front, right needle through front of left loop (back to front), yarn over, pull through, slip off
- **Bind off**: finish edge — knit 2 stitches, pass first over second, repeat. Don't bind off too tight (use larger needle for bind off row)

## Crochet Essentials
- **Materials**: yarn + hook (sizes B-S/ 2.25-15mm — choose hook size that matches yarn weight, recommendation on yarn label)
  - Ergonomic hooks: recommended for longer sessions (less hand pain)
  - Yarn labels: weight symbol (0-7), recommended hook size, gauge, yardage, dye lot (buy same dye lot for consistent color!)
- **Slip knot**: secure yarn on hook
- **Chain (ch)**: foundation of most projects — yarn over, pull through loop
- **Single crochet (sc)**: insert hook, yarn over, pull up loop (2 loops on hook), yarn over, pull through both — short, dense stitch
- **Half double crochet (hdc)**: yarn over, insert hook, yarn over, pull through (3 loops), yarn over, pull through all 3
- **Double crochet (dc)**: yarn over, insert hook, pull through (3 loops), yarn over pull through 2, yarn over pull through remaining 2 — taller, looser
- **Slip stitch (sl st)**: insert hook, yarn over, pull through loop AND the loop on hook — joining, moving stitch marker, decorative edge

## Reading Patterns
- **Abbreviations** (knit/crochet): K (knit), P (purl), CO (cast on), BO (bind off), inc (increase), dec (decrease), rep (repeat), st(s) (stitch(es)), yo (yarn over), RS/WS (right/wrong side), PM (place marker), SM (slip marker)
  - Crochet: ch (chain), sc (single crochet), hdc (half double), dc (double), tr (treble)
- **Gauge**: stitches + rows per 4 inches (10 cm) — CRITICAL for fit (make a swatch! wash it! measure before starting project!)
  - If your gauge doesn't match pattern: change needle/hook size (more stitches = smaller needle, fewer = larger)
  - Follow gauge exactly for garments. For blankets/scarves/shawls: gauge is less critical
- **Sizes**: patterns list multiple sizes with instructions in order (S, M, L, XL) — circle your size, follow throughout
- **Charts**: visual representation of stitch pattern — common for lace, cables, colorwork — read bottom to top, right to left for right-side rows

## Yarn Weights & Fibers
- **Lace (0)**: super fine, socks, shawls (requires thin needles/hooks)
- **Fingering/sock (1)**: socks, shawls, light garments
- **Sport (2)**: baby clothes, lightweight sweaters
- **DK/double-knitting (3)**: most common for garments — versatile, medium weight
- **Worsted/aran (4)**: thick sweaters, hats, blankets — also very common, works up quickly
- **Bulky (5)**: quick projects, hats, scarves, blankets — chunky yarn, big needles/hooks
- **Super-bulky (6+)**: arm knitting, very fast blankets, thick winter accessories
- **Fibers**: wool (warm, elastic, breathable, felts), cotton (cool, no stretch, holds shape, stiff), acrylic (cheap, hypoallergenic, washable, doesn't breathe), alpaca (very warm, soft, pricey, stretches), silk (luxurious, strong, drapey, expensive), merino (soft wool, non-itchy, great for next-to-skin), cashmere (luxurious, delicate, $$$)
  - For gifts: check care requirements (acrylic = machine washable, wool = hand wash)
  - Blends combine benefits: wool+acrylic (warm + affordable), cotton+acrylic (breathable + machine washable)

## Beginner Projects
- **Knitting**: scarf (garter stitch = knit every row, practice consistency), dishcloth (learn knit + purl), hat in the round (circular needles, increases, ribbing)
- **Crochet**: washcloth/dishcloth (single crochet squares), scarf (double crochet — fast, meditative), granny square blanket (individual squares, learn joining), beanie (simple hat worked flat or in round)

## Fixing Mistakes
- **Knitting**: dropped stitch (catch with crochet hook, pull up ladder rungs), twisted stitch (slip off, reorient, put back on left), frogging (rip out — "rip it, rip it"). Tink (unknit: insert left needle into loop below, slip stitch off right, pull yarn)
