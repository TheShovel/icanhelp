# Cooking Techniques Deep Dive

## Searing & Maillard Reaction

### Science
```
Maillard: Amino acids + reducing sugars + heat (280-330°F) → flavor compounds
Caramelization: Sugar oxidation (320°F+) → nutty, sweet flavors
Both create complexity, but different chemistry
```

### Perfect Sear
```
1. Dry surface completely (pat, air dry, salt early)
2. Heavy pan (cast iron, carbon steel, stainless)
3. High heat, thin oil film (smoke point appropriate)
4. Don't crowd (steam prevents browning)
5. Don't move until releases naturally
6. Deglaze for sauce
```

### Oil Smoke Points
```
Avocado: 520°F
Safflower: 510°F
Light olive: 465°F
Ghee: 485°F
Canola: 400°F
Grapeseed: 420°F
Peanut: 450°F

Unrefined coconut: 350°F
```

## Braising & Stewing

### Braise
```
Large cuts, ⅓-½ liquid, covered, low & slow (300°F oven or simmer)
Collagen → gelatin (160-180°F) = tender, silky texture
Best: Chuck, short ribs, brisket, shank, pork shoulder, lamb shank
```

### Stew
```
Small pieces, fully submerged, simmer (not boil)
Same collagen conversion
Best: Chuck, shoulder, stew meat
```

### Technique
```
1. Sear meat (flavor foundation)
2. Sauté aromatics in same pot
3. Deglaze (wine, stock, acid)
4. Return meat, add liquid to ½-⅔
5. Simmer gently (bubbles barely breaking)
6. Oven 300°F = even heat, no scorching
7. Cool in liquid (reabsorbs moisture)
8. Skim fat, reduce sauce if needed
```

## Roasting

### High Heat (425-475°F)
```
Vegetables: Caramelization, crisp edges
Chicken pieces: Crispy skin
Fish: Quick, moist interior
```

### Low & Slow (250-325°F)
```
Large roasts: Even cooking, tender
Prime rib, pork shoulder, turkey
Rest carryover: 10-15°F rise
```

### Technique
```
Dry surface → oil → season → wire rack on sheet pan
Don't crowd (steam)
Rotate pan halfway
Rest before carving (juices redistribute)
```

## Grilling

### Direct Heat
```
High heat, lid open
Burgers, steaks, chops, vegetables, fish
Sear → move to indirect if thick
```

### Indirect Heat
```
Heat on one side, food on other, lid closed
Roasts, whole chicken, ribs, pizza
Like oven but with smoke flavor
```

### Two-Zone Setup
```
Charcoal: Coals on one side
Gas: One burner on, others off
Sear direct → finish indirect
```

### Smoke
```
Wood chips/chunks on coals or smoker box
Soak chips 30 min (prolongs smoke)
Don't oversmoke (bitter)
Thin blue smoke > thick white smoke
```

## Frying

### Pan Fry (Shallow)
```
½" oil, 325-375°F
Breaded items: Flour → egg → crumbs (press gently)
Don't crowd (temp drops)
Drain on wire rack (not paper towels = soggy)
```

### Deep Fry
```
3-4" oil, 325-375°F
Thermometer essential
Batter: Cold liquid → hot oil = steam barrier = crisp
Double fry: Low temp cook through → high temp crisp
Oil maintenance: Filter, store cool/dark, discard when dark/foamy
```

### Temp Guide
```
French fries: 325°F (first) → 375°F (second)
Chicken: 325-350°F
Fish: 350-375°F
Donuts: 350-375°F
Tempura: 350-375°F
```

## Poaching

### Shallow Poach
```
½-⅔ liquid, simmer (180-190°F), covered
Fish, chicken breasts, fruit
Court bouillon: Water + wine + aromatics + acid
```

### Deep Poach
```
Fully submerged, bare simmer
Eggs (with vinegar), dumplings
```

### Technique
```
Liquid: Stock, wine, water + aromatics
Temp: 160-180°F (shimmer, no bubbles)
Time: Fish 8-10 min, chicken 15-20 min, eggs 3-4 min
```

## Steaming

### Methods
```
Steamer basket over pot
Bamboo steamer
Oven steam (Dutch oven + lid)
Microwave (covered with splash water)
```

### Applications
```
Vegetables: Retains nutrients, color
Fish: Gentle, moist
Dumplings/Buns: Traditional
Tamales: Masa steamed in husk
Puddings: Christmas pudding
```

### Technique
```
Water not touching food
Lid tight
Rapid boil = more steam
Don't overcrowd
```

## Sous Vide

### Principle
```
Precision temperature water bath
Vacuum sealed bag
Impossible to overcook (temp = doneness)
Pasteurization at lower temps (time dependent)
```

### Temps
```
Steak: 129°F (rare) - 135°F (medium)
Chicken breast: 145-150°F (juicy, safe)
Chicken thigh: 165°F (traditional) or 150°F (2+ hrs)
Salmon: 120°F (rare) - 130°F (medium)
Pork chop: 140°F
Eggs: 145°F (soft) - 167°F (hard)
Carrots: 185°F
```

### Process
```
1. Season, bag, vacuum seal
2. Water bath at target temp
3. Cook (1-48 hrs depending)
4. Sear for crust (dry well, high heat, fast)
```

## Emulsions

### Vinaigrette (Temporary)
```
3:1 oil : acid
Whisk vigorously or shake in jar
Mustard/emulsifier helps stabilize
```

### Mayonnaise (Permanent)
```
Egg yolk + acid + slow oil stream
Lecithin in yolk = emulsifier
Ratio: 1 yolk : ¾-1 cup oil
Rescue broken: New yolk + slow whisk broken in
```

### Hollandaise/Béarnaise
```
Egg yolks + acid + warm clarified butter
Double boiler, constant whisk
Rescue: Cold water or new yolk
```

### Pan Sauce (Deglaze)
```
Fond (browned bits) = flavor
Deglaze: Wine, stock, acid
Reduce → mount with cold butter (emulsion)
```

## Knife Skills - Advanced

### Oblique / Roll Cut
```
Roll vegetable ¼ turn, diagonal cut
Uneven surfaces = more sauce adhesion
Carrots, parsnips, zucchini
```

### Paysanne
```
Thin squares (½" × ½" × ⅛")
From batonnet, slice crosswise
Soups, garnishes
```

### Tournée
```
7-sided football shape
Root vegetables
Shows skill, even cooking
```

### Butterflying
```
Cut horizontally, not through, open like book
Chicken breast, pork chop, shrimp
Even thickness = even cooking
```

### Scoring
```
Shallow cuts in pattern
Fat cap (duck, pork): Render fat, crisp skin
Fish skin: Prevent curling
Bread: Controlled expansion

## Heat Management

### Carryover Cooking
```
Residual heat continues cooking
Rest 5-15 min (large roasts 20-30)
Temp rises 5-15°F
Pull before target temp
```

### Resting Meat
```
Juices redistribute (pressure equalizes)
Cut hot = juice runs out
Tent foil (don't wrap tight = steam)
```

### Pan Temperature Test
```
Water droplet: Dances = ready (Leidenfrost)
Oil: Shimmers, flows like water
Butter: Foam subsides = ready
```

## Flavor Development

### Deglazing
```
After sear, add liquid to hot pan
Scrape fond (browned bits) = flavor gold
Wine, stock, vinegar, water
Reduce → concentrate
```

### Reduction
```
Simmer uncovered → water evaporates
Flavor concentrates, thickens naturally
Don't boil hard (bitter)
Finish with butter (shine, body)
```

### Blooming Spices
```
Whole spices in hot oil 30-60 sec
Releases essential oils
Don't burn (bitter)
Ground spices: Add later (burn faster)
```

### Finishing
```
Acid: Lemon, vinegar (brightens)
Fresh herbs: Parsley, cilantro, basil, dill, mint
Fat: Butter, olive oil (mouthfeel)
Salt: Flaky finishing salt
Texture: Toasted nuts, seeds, breadcrumbs
```

## Common Problems & Fixes

| Problem | Cause | Fix |
|---------|-------|-----|
| Steamed not seared | Wet surface, crowded pan, low heat | Dry, space, hotter |
| Burnt outside, raw inside | Too high heat | Lower, finish in oven |
| Tough braised meat | Boiled not simmered | Lower temp, longer time |
| Broken sauce | Too hot, added fat too fast | Cool, whisk slowly |
| Soggy fried food | Oil too cool, crowded | Hotter oil, smaller batches |
| Dry chicken breast | Overcooked | Thermometer, brine, pound thin |
| Mushy vegetables | Overcooked | Blanch/shock, roast high heat |
| Bitter sauce | Burnt fond, burnt garlic | Deglaze carefully, add garlic late |
| Greasy braise | Not skimmed | Chill, lift solid fat; or skim hot |
| Curdled dairy | Boiled, acid + heat | Temper, add at end, low heat |
