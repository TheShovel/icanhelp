# Cooking Fundamentals

## Knife Skills

### Basic Cuts
```
Julienne:     Matchsticks (3mm x 3mm x 5cm)
Batonnet:     Thick sticks (6mm x 6mm x 6cm)
Dice:         Cubes
  Brunoise:   3mm (from julienne)
  Small:      6mm
  Medium:     1cm
  Large:      2cm
Mince:        <2mm, as fine as possible
Chiffonade:   Leafy strips (roll leaves, slice thin)
```

### Knife Safety
```
- Sharp knife = safe knife (less force = more control)
- Claw grip: Fingertips tucked, knuckles guide blade
- Cutting board stable (damp towel underneath)
- Cut away from body
- Don't catch falling knife
- Store in block or on magnetic strip
```

### Knife Types
```
Chef's (8-10"):  90% of tasks
Paring (3-4"):   Peeling, detail work
Bread (serrated): Bread, tomatoes, citrus
Boning:          Meat/fish fabrication
Utility:         Between paring and chef's
```

## Heat Transfer Methods

### Dry Heat
```
Sauté:        High heat, little fat, toss frequently
Pan-fry:      Medium-high, ½" fat, don't crowd
Deep-fry:     325-375°F, fully submerged
Roast:        Oven, uncovered, dry heat (400°F+)
Bake:         Oven, lower temp (300-375°F)
Broil:        Direct top heat (500°F+)
Grill:        Direct heat from below
```

### Moist Heat
```
Blanch:       Brief boil, shock in ice water
Boil:         212°F, vigorous bubbles
Simmer:       185-205°F, gentle bubbles
Poach:        160-180°F, no bubbles, shimmer
Steam:        212°F+, vapor heat
Braise:       Sear, then slow cook in liquid (½ covered)
Stew:         Small pieces, fully submerged, simmer
```

### Combination
```
Sear then roast
Braise (sear + moist)
```

## Flavor Building

### Mirepoix (French)
```
2 parts onion : 1 part carrot : 1 part celery
Sauté in butter/oil until soft (not browned)
Base for stocks, soups, sauces, braises
```

### Sofrito (Spanish/Latin)
```
Onion, garlic, pepper, tomato
Cooked slow in olive oil
```

### Holy Trinity (Cajun)
```
2 parts onion : 1 part celery : 1 part green bell pepper
```

### Battuto (Italian)
```
Onion, carrot, celery, sometimes garlic/parsley
Finely minced, sautéed in oil/butter
```

### Aromatics by Cuisine
```
Indian:      Onion, garlic, ginger, chili, spices (cumin, coriander, turmeric)
Thai:        Garlic, chili, lemongrass, galangal, kaffir lime, fish sauce
Chinese:     Ginger, garlic, scallion, soy, rice wine
Mexican:     Onion, garlic, chili, tomato, cumin, oregano
Mediterranean: Garlic, onion, herbs (oregano, thyme, rosemary), lemon
```

### Layering Flavor
```
1. Aromatics (fat + vegetables)
2. Spices (toast briefly in fat)
2. Main ingredients
3. Liquid (stock, wine, water)
4. Herbs (hardy early, delicate late)
5. Acid (finish: lemon, vinegar, wine)
6. Fat (finish: butter, oil)
7. Season (salt throughout, pepper late)
```

## Salt

### Types
```
Kosher:      Large flakes, easy to pinch, no additives (Diamond Crystal = less dense)
Sea salt:    Flaky, finishing (Maldon, Fleur de Sel)
Table:       Fine, iodized, anti-caking (densest)
Himalayan:   Pink, trace minerals, finishing
```

### Salting Strategy
```
Meat:        1% by weight, 40 min - 24 hrs before (dry brine)
Vegetables:  During cooking (draws water)
Pasta water: "Like the sea" (1-2% salinity)
Beans:       Soak in salted water, cook in salted
Stock:       Lightly at end (reduces)
Finishing:   Flaky salt on plated dish
```

## Stocks & Broths

### White Stock (Chicken/Veal/Fish)
```
Bones (not roasted) + mirepoix + herbs
Simmer 3-4 hrs (chicken), 6-8 hrs (veal), 30 min (fish)
Skim impurities
```

### Brown Stock (Beef/Veal/Game)
```
Roast bones 400°F 45 min → deglaze pan → simmer 8-12 hrs
Deeper flavor, color
```

### Vegetable Stock
```
Onion, carrot, celery, leek, fennel, mushroom, parsley, thyme, bay, peppercorn
Avoid: Brassicas (cabbage, broccoli), starchy veg, beets
Simmer 45-60 min only (bitter if longer)
```

### Quick Stock (20 min)
```
Store-bought low-sodium + aromatics + simmer 20 min
Better than plain water
```

## Sauces - Mother Sauces (French)

### Béchamel (White)
```
Roux (butter + flour) + milk + nutmeg
Base for: Mornay (cheese), Soubise (onion), Nantua (crayfish)
```

### Velouté (Blond)
```
Roux + white stock (chicken/fish/veal)
Base for: Allemande (egg/cream), Supreme (cream/mushroom), Normande (fish/cream)
```

### Espagnole (Brown)
```
Brown roux + brown stock + tomato + mirepoix
Base for: Demi-glace (reduced ½), Bordelaise (wine/shallot), Chasseur (mushroom/tomato)
```

### Tomate
```
Tomato + mirepoix + stock + pork + herbs
Base for: Creole (pepper/onion), Provençale (garlic/olive/herbs)
```

### Hollandaise (Emulsion)
```
Egg yolks + clarified butter + lemon + cayenne
Warm emulsion (bain-marie)
Base for: Béarnaise (tarragon/shallot/peppercorn), Maltaise (blood orange), Mousseline (whipped cream)
```

## Eggs

### Cooking Methods
```
Boiled:    Start cold water, boil, cover, off heat
  Soft:    4-6 min (runny yolk)
  Medium:  7-9 min (jammy)
  Hard:    10-12 min (fully set)
Poached:   180°F water + vinegar, 3-4 min, fresh eggs
Fried:     Butter/oil, medium-low
  Sunny:   Unflipped, runny yolk
  Over-easy: Flipped, runny yolk
  Over-hard: Flipped, set yolk
Scrambled: Low heat, constant stir, off heat before done
Omelet:    High heat, tilt pan, roll (French) or fold (American)
```

### Egg Science
```
Whites set:  144-149°F
Yolks set:   149-158°F
Custard:     175-185°F (don't boil or curdles)
```

## Pasta

### Ratio
```
1 lb pasta : 4-6 quarts water : 1-2 tbsp salt
```

### Technique
```
1. Boil water, salt heavily
2. Add pasta, stir immediately
3. Cook 1-2 min less than package
4. Reserve 1 cup pasta water
4. Drain, add to sauce with pasta water
5. Finish 1-2 min in sauce (emulsifies)
```

### Shapes & Sauces
```
Long (spaghetti):      Smooth, oil/tomato/cream
Tubes (penne):         Chunky, meat/vegetable
Ribbons (fettuccine):  Cream/cheese
Shaped (farfalle):     Pesto, light cream
Tiny (orzo):           Soups, salads
Stuffed (ravioli):     Butter/sage, light tomato
```

## Rice

### Ratios (Absorption Method)
```
Long grain (basmati, jasmine):  1 : 1.5 (rice : water)
Medium grain:                   1 : 2
Short grain (sushi, arborio):   1 : 1.25
Brown rice:                     1 : 2.5
```

### Methods
```
Pilaf:       Sauté rice in fat → add liquid → simmer covered 18 min → rest 10 min
Steamed:     Rinse → soak 30 min → drain → steam 20 min
Risotto:     Sauté aromatics → toast rice → add wine → add hot stock ladle by ladle → finish with butter/cheese
```

## Beans & Legumes

### Dried Beans
```
Soak: 8-12 hrs (or quick soak: boil 2 min, soak 1 hr)
Cook: Simmer 1-3 hrs until tender
Salt: Add at beginning (helps skin stay intact)
Acid: Add at end (prevents softening)
```

### Quick-Cook (No Soak)
```
Lentils (red 15 min, green/brown 25 min)
Split peas (30-40 min)
Black-eyed peas (45-60 min)
```

## Vegetables

### Blanching (Green Veg)
```
1. Boil salted water
2. Add veg, 1-3 min until bright green
3. Shock in ice water
4. Drain, reheat in butter/oil before serving
```

### Roasting (Root Veg, Brassicas)
```
425-450°F, single layer, don't crowd
Toss with oil, salt, spices
Flip halfway
Caramelization = flavor
```

### Sautéing (Tender Veg)
```
High heat, fat shimmering
Don't crowd (steams instead)
Cook in batches
Finish with acid/herbs
```

## Meat

### Doneness Temperatures
```
Beef/Lamb/Veal:
  Rare:       125°F (rest to 130°F)
  Medium-rare: 130°F (rest to 135°F)
  Medium:     135°F (rest to 140°F)
  Medium-well: 145°F (rest to 150°F)
  Well:       155°F+

Pork:         145°F (rest 3 min)
Chicken/Turkey: 165°F (breast), 175°F (thigh)
Ground meats: 160°F
Fish:         145°F (or opaque/flakes)
```

### Resting
```
Critical for juiciness
Rest 5-10 min (small), 15-20 min (large roast)
Carryover cooking: +5-10°F
Tent with foil (don't wrap tight)
```

### Searing
```
Dry surface (pat dry)
Hot pan, thin oil film
Don't move until releases
Deglaze pan for sauce
```

## Baking Basics

### Measuring
```
Flour: Spoon into cup, level off (don't scoop = 20% more)
Weigh: Grams = accurate (1 cup AP flour = 120-125g)
Liquids: Clear measuring cup, eye level
```

### Temperature
```
Butter: Room temp = 65-67°F (dents but holds shape)
Eggs: Room temp = 30 min out
Oven: Preheat 20 min, thermometer verify
```

### Leaveners
```
Baking soda: Base, needs acid (buttermilk, yogurt, vinegar, cocoa)
Baking powder: Double-acting (acid + base), works twice
Yeast: Alive, needs warmth (105-115°F), food, time
```

### Common Ratios
```
Cookies:  3 flour : 2 fat : 1 sugar (by weight)
Cake:     1 flour : 1 sugar : 1 fat : 1 egg
Pie crust: 3 flour : 2 fat : 1 water
Bread:    5 flour : 3 water : 1 yeast : 2% salt
```

## Kitchen Safety

### Fire
```
Grease fire: Lid on, turn off heat, NO WATER
Oven fire: Keep door closed, turn off
Extinguisher: Class K (kitchen), PASS method
```

### Cuts
```
Apply pressure, elevate
Deep/won't stop bleeding → ER
```

### Burns
```
Cool under running water 10-20 min
Don't ice, butter, toothpaste
Blistered/severe → medical attention
```
