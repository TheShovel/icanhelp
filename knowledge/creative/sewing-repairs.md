# Basic Sewing Repairs

## Tools
- Hand sewing needles: sharp (general, medium) vs ballpoint (knits — doesn't split fibers). Embroidery/crewel (larger eye, for thicker thread). Betweens/quilting (short, small eye, for tiny stitches). Needles: sizes 1-12 (higher = finer). For general: 5-8
- Thread: all-purpose polyester (strong, slight stretch, good for most repairs). Cotton (for natural fabrics only — weaker). Silk (delicate, expensive). Denim/jeans (thick, strong, for heavy fabric). Nylon (stretchy, for outdoor gear). Beeswax: run thread through wax to prevent tangling + strengthen
- Thimble: protects finger when pushing needle through thick fabric (denim, leather). Use on middle finger of dominant hand. Metal > leather > plastic for general. Quilting (leather/soft)
- Seam ripper: for removing stitches. Use carefully (don't cut fabric). Use curved edge to cut stitches, not blade tip into fabric

## Common Repairs

### Sew on a Button
1. Thread needle, knot end. Mark button position (original thread marks usually visible)
2. Push needle from underside of fabric through one buttonhole (start with a knot on the underside, hide under button later)
3. Cross over to other hole (for 4-hole: parallel lines or X pattern). Space: leave slack for shank (with 4-hole: cross over the button to create space)
4. For thicker fabrics: create thread shank (wrap thread around stitches between button + fabric 4-6 times before knotting). Shank = space for buttonhole layer to fit
5. Secure: small back stitches on fabric surface under button
6. For coat buttons: use heavy-duty thread or button thread (waxed). X pattern for 4-hole use for jeans buttons

### Fix a Dropped Hem
1. Iron hem flat (create a crisp fold). Turn under raw edge 1/4-1/2" (to hide raw edge). Pin in place (perpendicular to hem edge)
2. Stitch: slip stitch (invisible — pick up one thread from garment, then from hem fold, 1/4" apart, repeat). Running stitch (visible, quicker). Catch stitch (for stretchy fabric — crisscross)
3. Shortcut: fusible hem tape (iron-on, quick, temporary — not permanent through washing). Good for: quick fix, travel, non-stress areas. Best for lightweight fabrics
4. Mending a ripped hem (torn out hem in denim, trousers): can use iron-on fusible interfacing behind tear + topstitching. Then hand stitch the hem back up + reinforce area

### Patch a Hole
- **Iron-on patch**: peel-and-stick, quick fix (laundry risk: may peel off in hot water + dryer). Better: iron-on + sew around edges (Sewing through iron-on works! Use a new needle, not blunt)
- **Sew-in patch**: cut patch larger than hole (1-2" on each side). Pin on inside of garment. Stitch around edges (zigzag or straight stitch close to edge). Use matching thread (or contrasting for decorative). For denim: use denim patch + heavy thread + reinforce with denim needle
- **Darning**: weave stitches across hole (like weaving basket). For knit fabrics (socks): use darning mushroom (smooth curved wooden shape under stretched fabric). Weave thread back + forth across hole, then perpendicular (fill the hole). Sock darning: use matching thread/yarn. Distribute tension evenly

### Repair a Seam
1. Turn garment inside-out. Position ripped seam edges together. Match original stitch line
2. Machine stitch (backstitch beginning + end to lock) over original seam, extending 1" beyond rip on both sides. Use 2mm stitch length for permanence
3. Hand stitch: backstitch (strongest) or running stitch (quicker). For de-laminated seam (edge turned under): turn under raw edge, re-stitch along the edge
4. Fix serged seam (overlock seam ripped): use zigzag stitch if no serger

### Fix a Zipper
- **Separated zipper (bottom came apart)**: sometimes chain just came apart. If zipper tape is intact: manually re-thread zipper pull (align teeth, thread slider back on from top). Use pliers to gently squeeze the bottom stop top of opening STOPS? Actually separate the bottom stop of zipper tape from the slider (slides off). Reinsert tape into top of slider, pull up. For broken zipper pull: replace pull (some are interchangeable, some require buying new zipper tape)
- **Broken zipper pull**: zipper is intact but pull broke (the part you hold). New pulls available as universal replacements (slide on from top, can't exit bottom). Metal key ring can serve as temporary replacement
- **Stuck zipper**: rub graphite (pencil) or bar soap on teeth. Do NOT use oil (stains fabric, attracts dirt). Gently work zipper up+down. If the fabric is caught in the teeth: do NOT force zipper — carefully extract fabric out with pin or tweezers. If no success: replace zipper (seam ripper, new zipper, re-stitch)
- **Replace entire zipper**: seam ripper to remove old, iron to remove crease marks, pin new zipper in place, baste, machine stitch (zipper foot — can use standard or invisible foot. For lapped zipper: stitch on one side, fold back and stitch on other side. For invisible zipper: special foot, press coils flat before sewing). For a simple change: cut new zipper to length (metal zipper: cut new one to fit, remove old and stitch directly, remove metal stops (transfer stops or replace with new stop))

## Quick Fixes
- **Loose button**: already reinforced with small secure back stitch. Add a drop of clear nail polish or superglue on thread to keep tight? Not fabric-safe — use Fray Check instead (fabric glue prevents thread from raveling)
- **Snap/velcro replacement**: snap pliers (anvil + die, $10-20 kit). Screw-on snaps (no tool needed). Velcro: sew or iron-on (peel + stick) — best on cuffs, hems, and slits. Stitch securely for high-use garments
- **Drawstring replacement**: use safety pin to thread new drawstring through casing. Attach safety pin to one end, feed through. Make new end stop: knot, fold over, stitch, or use a large bead/chord tip
- **Elastic replacement**: open casing, remove old elastic (it's dry-rotted or stretched). Cut new elastic to size (waist measurement - 2" for stretch). Attach safety pin to one end, thread through, overlap + stitch ends. Close casing (stitch casing opening). Common replacement: 1/4 to 1" elastic for pants, shorts, and skirt waists
- **Belt loop repair**: topstitch broken loop back in place (use a triple stitch/bar tack for strength). Reinforce original attachment point
