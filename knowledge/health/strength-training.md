# Strength Training Fundamentals

## Principles

### Progressive Overload
```
Increase demand over time:
- Weight (intensity)
- Reps/sets (volume)
- Frequency
- Density (work/rest ratio)
- Range of motion
- Tempo (time under tension)
```

### Specificity (SAID Principle)
```
Specific Adaptation to Imposed Demands
Train the movement/energy system you want to improve
```

### Recovery
```
Muscle grows during rest, not training
48-72 hours between same muscle group
Sleep: 7-9 hours
Protein: 1.6-2.2g/kg bodyweight
Deload: Every 4-8 weeks (reduce volume 50%)
```

## Movement Patterns

### 6 Fundamental Patterns
```
1. Squat:      Knee dominant (squat, lunge, step-up)
2. Hinge:      Hip dominant (deadlift, RDL, swing, good morning)
3. Push:       Horizontal (bench, pushup) / Vertical (overhead press)
4. Pull:       Horizontal (row) / Vertical (pull-up, lat pulldown)
5. Carry:      Loaded carries (farmer, suitcase, waiter)
6. Core:       Anti-extension, anti-rotation, anti-lateral flexion
```

### Exercise Selection Priority
```
Compound (multi-joint) first → Isolation (single-joint) after
Bilateral → Unilateral
Free weights → Machines (for stability/accessory)
```

## Programming Variables

### Rep Ranges & Adaptations
```
1-5 reps @ 85%+:  Maximal strength (neural)
6-12 reps @ 65-85%: Hypertrophy (muscle size)
12-20+ reps @ <65%: Muscular endurance
Power: 1-5 reps @ 30-70% max velocity
```

### Sets
```
Beginner: 2-3 sets per exercise
Intermediate: 3-4 sets
Advanced: 4-6 sets
Per muscle/week: 10-20 sets (hypertrophy)
```

### Rest Periods
```
Strength/Power: 3-5 min
Hypertrophy:    1-2 min
Endurance:      30-60 sec
Supersets:      Minimal (alternate movements)
```

### Tempo
```
Eccentric: 2-4 sec (control)
Isometric: 0-2 sec (pause)
Concentric: Explosive (X) or 1-2 sec
Notation: 3-0-X-0 = 3 sec down, no pause, explosive up, no pause
```

## Program Design

### Frequency Options
```
Full Body:     3x/week (A/B alternating)
Upper/Lower:   4x/week (Mon/Thu Upper, Tue/Fri Lower)
Push/Pull/Legs: 6x/week (or 3x with 2x frequency)
Body Part Split: 5-6x/week (advanced only)
```

### Beginner Template (3x/week Full Body)
```
A: Squat 3×5, Bench 3×5, Row 3×8, RDL 3×8, Plank 3×30s
B: Deadlift 1×5, OHP 3×5, Lunge 3×10, Pull-up 3×max, Side plank 3×20s
Alternate A/B/A, B/A/B
Linear progression: Add 2.5-5 lbs each session
```

### Intermediate Template (4x/week Upper/Lower)
```
Upper A: Bench 4×6, Row 4×8, Incline DB 3×10, Face pull 3×15, Curl 3×12
Lower A: Squat 4×6, RDL 3×8, Leg press 3×10, Calf 4×12, Core
Upper B: OHP 4×6, Pull-up 4×8, Dip 3×10, Rear delt 3×15, Tricep 3×12
Lower B: Deadlift 3×5, Front squat 3×8, Lunge 3×10, Hip thrust 3×12, Core
```

### RPE/RIR (Auto-regulation)
```
RPE 10: Max effort (0 reps in reserve)
RPE 9:  1 rep left
RPE 8:  2 reps left
RPE 7:  3 reps left
RPE 6:  4 reps left
Target: RPE 7-9 for most sets
RIR = Reps in Reserve (same scale)
```

## Key Lifts - Technique Cues

### Squat
```
Stance: Shoulder-width, toes out 15-30°
Bar: High bar (traps) or low bar (rear delts)
Brace: Big breath, 360° expansion
Descent: Hip/knee simultaneously, knees track toes
Depth: Hip crease below knee (parallel)
Ascent: Drive traps into bar, push floor away
```

### Deadlift
```
Stance: Hip-width, bar over mid-foot
Grip: Double overhand → mixed → hook
Setup: Shins touch bar, hips high, chest up
Pull: Push floor away, bar stays close
Lockout: Squeeze glutes, don't hyperextend
Lower: Hinge back, control down
```

### Bench Press
```
Setup: Eyes under bar, feet planted, arch slight
Grip: Wrists stacked, bar on heel of palm
Unrack: Straight arms, pull bar to chest
Descent: 45° elbow tuck, touch lower chest
Press: Drive feet, push back toward rack
```

### Overhead Press
```
Stance: Hip-width, glutes squeezed
Grip: Just outside shoulders, forearms vertical
Path: Slightly back, around face
Lockout: Head through "window", biceps by ears
```

### Row (Barbell/Dumbbell)
```
Hinge: ~45°, neutral spine
Pull: Elbow to hip pocket, not high
Squeeze: Scapula together at top
Lower: Controlled, full stretch
```

### Pull-Up
```
Grip: Slightly wider than shoulders
Start: Dead hang, shoulders depressed
Pull: Chest to bar, elbows down/back
Lower: Controlled, full extension
Progress: Band assist → negative → weighted
```

## Accessory Categories

### Shoulder Health
```
Face pulls, band pull-aparts, YTWL, external rotation
Scapular push-ups, wall slides
2-3 sets × 15-20, 2-3x/week
```

### Core (Anti-Movement)
```
Anti-extension: Plank, dead bug, ab wheel
Anti-rotation: Pallof press, bird dog
Anti-lateral flexion: Side plank, suitcase carry
Flexion: Hanging leg raise, cable crunch (optional)
```

### Arms
```
Biceps: Curl variations (hammer, incline, preacher)
Triceps: Pushdown, overhead extension, dip, close-grip bench
Forearms: Wrist curl, reverse curl, farmer carry
```

### Calves
```
Standing (gastroc): Straight knee
Seated (soleus): Bent knee
High frequency, full ROM, pause at bottom
```

## Common Issues & Fixes

| Issue | Likely Cause | Fix |
|-------|-------------|-----|
| Knee cave (squat) | Weak glute med | Banded squats, clamshells, wider stance |
| Butt wink | Hip mobility/control | Ankle mobility, goblet squat, reduce depth |
| Rounded back (deadlift) | Weak posterior chain | RDL, back extension, lighter weight |
| Shoulder pain (bench) | Flared elbows, no retraction | 45° elbow tuck, scapular squeeze, floor press |
| Elbow pain (pull) | Overuse, grip width | Neutral grip, fat grips, reduce volume |
| Wrist pain (press) | Extension under load | Wrist wraps, neutral grip DB, push-up handles |
| Lower back pain | Weak core/glutes, poor hinge | Bird dog, dead bug, glute bridge, RDL |

## Nutrition for Strength

### Calories
```
Maintenance: Bodyweight × 14-16
Surplus: +200-500 (gain 0.25-0.5% bw/week)
Deficit: -300-500 (lose 0.5-1% bw/week)
```

### Protein
```
1.6-2.2g/kg (0.7-1g/lb) bodyweight
Spread across 3-5 meals
Leucine threshold: ~3g per meal (30-40g protein)
Sources: Meat, fish, eggs, dairy, whey, soy, legumes
```

### Carbs/Fats
```
Carbs: 3-5g/kg (fuel training)
Fats: 0.8-1g/kg (hormone health)
Fiber: 25-35g
Water: 3-4L + during training
```

### Timing
```
Pre: Carbs + protein 1-3 hrs before
During: Water, electrolytes if >90 min
Post: Protein + carbs within 2 hrs (anabolic window wider than thought)
```

## Recovery Tools

### Sleep
```
7-9 hours, consistent schedule
Cool (65-68°F), dark, quiet
No screens 1 hr before
Magnesium glycinate, glycine, tart cherry if needed
```

### Active Recovery
```
Walking, light cycling, swimming, yoga
Zone 2 cardio (conversational pace)
Mobility flow
```

### Modalities (Evidence-Based)
```
Foam rolling: Acute ROM, reduce DOMS
Massage: Recovery perception
Compression: May help recovery
Cold exposure: Acute inflammation, not for hypertrophy
Sauna: Cardiovascular, possibly recovery
```

## Special Populations

### Older Adults (50+)
```
Higher frequency, lower intensity
More machines for stability
Emphasis: Power (fall prevention), balance, bone density
Protein: 1.5-2g/kg
Longer warm-up, joint-friendly variations
```

### Women
```
Same principles, similar relative potential
Higher volume tolerance
Menstrual cycle: Follicular = higher intensity, Luteal = more recovery
Pregnancy: Doctor clearance, avoid supine after 1st trimester, no Valsalva
Postpartum: Pelvic floor first, progressive return
```

### Youth (Under 18)
```
Focus: Technique, fun, general athleticism
Bodyweight → light external load
No max attempts
Supervision required
Growth plates open → avoid axial loading excess
```

## Competition Prep (Powerlifting)

### Peaking
```
Volume ↓, Intensity ↑ over 8-12 weeks
Specificity ↑ (comp equipment, commands)
Openers: 90-92% 1RM (9/10 make rate)
Attempts: 1st = confident, 2nd = PR, 3rd = max
Meet day: Eat normal, hydrate, warm-up progressive
```
