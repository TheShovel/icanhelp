# Health Fundamentals

## Preventive Care Schedule (Adults)

### Annual
- [ ] Physical exam / wellness visit
- [ ] Blood pressure check
- [ ] BMI calculation
- [ ] Depression screening
- [ ] Alcohol/drug use screening
- [ ] Tobacco use counseling
- [ ] Immunization review
- [ ] Medication review

### Age-Based Screenings

#### 18-39
- [ ] Cholesterol: Every 5 years (or more if risk factors)
- [ ] Skin exam: Monthly self-check, yearly if high risk
- [ ] Testicular exam: Monthly self-check (men)
- [ ] Cervical cancer (women): Pap every 3 years (21-29), Pap+HPV every 5 years (30-65)
- [ ] STI screening: Based on risk factors
- [ ] Dental: Every 6 months
- [ ] Vision: Every 2 years (or yearly if contacts/glasses)

#### 40-49
- [ ] Mammogram (women): Every 1-2 years (discuss with doctor)
- [ ] Colorectal cancer: Start at 45 (colonoscopy every 10, FIT yearly, Cologuard every 3)
- [ ] Diabetes screening: Every 3 years (or earlier if overweight/risk factors)
- [ ] Eye exam: Every 2-4 years (baseline at 40)
- [ ] Prostate (men): Discuss PSA at 45-50

#### 50-64
- [ ] Mammogram: Every 1-2 years
- [ ] Colorectal: Continue screening
- [ ] Lung cancer: Annual LDCT if 50-80, 20 pack-year history, current/quit <15 years
- [ ] Osteoporosis (women): DEXA at 65, earlier if risk factors
- [ ] Shingles vaccine: 2 doses (Shingrix) at 50+
- [ ] Pneumococcal: At 65 (or earlier if risk factors)

#### 65+
- [ ] All above continuing
- [ ] Fall risk assessment: Annually
- [ ] Cognitive screening: Annually
- [ ] Hearing test: Every 1-2 years
- [ ] Vision: Annually
- [ ] Flu: Annually
- [ ] COVID: Per current guidance
- [ ] RSV: Single dose (60+)
- [ ] Pneumococcal: PCV20 or PCV15+PPSV23

### Vaccines (Adults)
```
Flu:              Annually (Sept-Oct ideal)
COVID-19:         Per current CDC guidance
Tdap:             Once, then Td/Tdap every 10 years
Shingles (Shingrix): 2 doses, 2-6 months apart (50+)
HPV:              Through age 26 (shared decision 27-45)
Pneumococcal:     PCV20 (65+) or PCV15+PPSV23
Hepatitis A/B:    If risk factors (travel, medical, lifestyle)
MMR:              If born 1957+ without immunity
Varicella:        If no immunity
Meningococcal:    If risk factors (college, military, travel)
RSV:              60+ (single dose)
```

## Nutrition Basics

### Macronutrient Targets (General)
```
Protein:    0.8-1.2g per kg body weight (1.6-2.2g/kg if strength training)
Carbs:      45-65% of calories (focus on fiber, whole foods)
Fat:        20-35% of calories (emphasize omega-3, limit trans/saturated)
Fiber:      25g women / 38g men (from whole foods)
Water:      ~3.7L men / ~2.7L women (includes food)
```

### Plate Method (Simple)
```
½ plate: Non-starchy vegetables
¼ plate: Lean protein
¼ plate: Complex carbs / starchy veg
+ Healthy fat (olive oil, nuts, avocado)
+ Fruit on side
```

### Key Principles
```
1. Eat mostly whole, minimally processed foods
2. Prioritize protein at each meal
3. 30g fiber daily (vegetables, fruits, legumes, whole grains)
4. Limit added sugar (<25g women, <36g men)
5. Limit sodium (<2300mg, ideally <1500mg)
6. Cook at home more often
7. Eat mindfully (no screens, chew slowly)
8. Stay hydrated (urine pale yellow)
```

### Common Deficiencies
```
Vitamin D:    Fatty fish, fortified dairy, sunlight, supplement (1000-2000 IU)
B12:          Animal products, fortified foods, supplement (vegan/vegetarian)
Iron:         Red meat, spinach, lentils (pair with vitamin C)
Magnesium:    Nuts, seeds, leafy greens, dark chocolate
Omega-3:      Fatty fish (salmon, sardines), walnuts, flax, supplement
Potassium:    Bananas, potatoes, beans, spinach
```

## Exercise Guidelines

### Minimum (WHO/CDC)
```
Aerobic: 150 min moderate OR 75 min vigorous per week
Strength: 2+ days per week (all major muscle groups)
Balance/Flexibility: Daily (especially 65+)
```

### Optimal
```
Aerobic: 300 min moderate OR 150 min vigorous
Strength: 3-4 days per week
Mobility: Daily 10-15 min
```

### Intensity Guide
```
Moderate:  Can talk, not sing (brisk walk, bike <10mph, doubles tennis)
Vigorous:  Can't say more than few words (run, swim laps, bike >10mph, singles tennis)
RPE Scale: 1-10 (moderate=5-6, vigorous=7-8)
Heart Rate: Moderate = 64-76% max HR, Vigorous = 77-93%
Max HR ≈ 220 - age
```

### Strength Training Basics
```
Compound movements: Squat, hinge, push, pull, carry, lunge
Frequency: 2-4x/week
Sets: 3-5 per exercise
Reps: 5-12 (strength 5-8, hypertrophy 8-12, endurance 12+)
Progressive overload: Increase weight, reps, sets, or decrease rest
Rest: 2-3 min (strength), 60-90s (hypertrophy), 30-60s (endurance)
```

### Sample Weekly Split
```
Option A (3-day full body):
Mon: Full body strength
Tue: Cardio + mobility
Wed: Full body strength
Thu: Cardio
Fri: Full body strength
Sat: Active recovery (walk, yoga)
Sun: Rest

Option B (4-day upper/lower):
Mon: Upper strength
Tue: Lower strength
Wed: Cardio + core
Thu: Upper strength
Fri: Lower strength
Sat: Cardio + mobility
Sun: Rest
```

## Sleep Hygiene

### Target: 7-9 hours

### Environment
```
Temperature: 65-68°F (18-20°C)
Darkness:    Blackout curtains, eye mask
Noise:       White noise, earplugs
Bed:         Comfortable mattress/pillow, replace 7-10 years
Use:         Bed = sleep + sex only
```

### Routine
```
Consistent: Same wake time daily (even weekends)
Wind-down:  30-60 min before bed (no screens)
Light:      Bright AM light (10 min), dim PM light
Caffeine:   Cut off 8-10 hours before bed
Alcohol:    Avoid 3+ hours before (fragments sleep)
Exercise:   Finish 3+ hours before bed
Naps:       <20 min, before 3 PM
```

### Supplements (Evidence-Based)
```
Magnesium glycinate: 200-400mg (relaxation)
Melatonin: 0.5-3mg (circadian shift, short-term)
Glycine: 3g (sleep quality)
L-theanine: 200mg (relaxation)
Apigenin: 50mg (chamomile derivative)
```

## Stress Management

### Acute (In the Moment)
```
Box breathing: 4-4-4-4 (inhale-hold-exhale-hold)
4-7-8 breathing: Inhale 4, hold 7, exhale 8
Physiological sigh: Double inhale, long exhale
Cold exposure: Face in cold water, cold shower
Grounding: 5-4-3-2-1 senses
Progressive muscle relaxation
```

### Chronic (Lifestyle)
```
Meditation: 10-20 min daily (app: Headspace, Waking Up, Insight Timer)
Nature:     120 min/week minimum (parks, hiking)
Social:     Meaningful connection daily
Hobbies:    Non-productive enjoyment
Boundaries: Say no, delegate, disconnect
Therapy:    CBT, ACT, DBT if needed
```

## Common Conditions - Quick Reference

### Hypertension (High BP)
```
Normal:     <120/<80
Elevated:   120-129/<80
Stage 1:    130-139 / 80-89
Stage 2:    ≥140 / ≥90
Crisis:     >180 / >120 (emergency)
Lifestyle:  DASH diet, <1500mg Na, exercise, weight loss, limit alcohol
```

### Type 2 Diabetes
```
Normal:     A1C <5.7%, Fasting <100
Prediabetes: A1C 5.7-6.4%, Fasting 100-125
Diabetes:   A1C ≥6.5%, Fasting ≥126
Management: Carb control, exercise, weight loss, metformin first-line
```

### High Cholesterol
```
Total:      <200 desirable
LDL:        <100 optimal (<70 if high risk)
HDL:        >60 protective, <40 risk
Triglycerides: <150
Non-HDL:    <130
```

### Red Flags (Go to ER)
```
Chest pain/pressure (especially with exertion)
Shortness of breath at rest
Stroke signs: BE FAST (Balance, Eyes, Face, Arm, Speech, Time)
Severe abdominal pain
Head injury with confusion/vomiting
Severe bleeding
Suicidal intent with plan
```

## Medication Safety

### Before Starting
```
Know: Name, dose, frequency, duration, purpose
Ask: Side effects, interactions, monitoring, alternatives
Check: Allergies, current meds (including OTC/supplements)
```

### Adherence
```
Pill organizer (weekly)
Phone alarms
Pair with habit (brush teeth)
Auto-refill pharmacy
Keep list updated (app or wallet card)
```

### Common Interactions
```
Grapefruit: Statins, some BP meds, antihistamines
Alcohol:    Most CNS depressants, metformin, antibiotics
NSAIDs:     Blood thinners, ACE inhibitors, lithium
Supplements: St. John's Wort (many), Vitamin K (warfarin)
```

### CPR (Adult)
```
1. Check responsiveness, call 911
2. 30 compressions (2" deep, 100-120/min)
3. 2 rescue breaths (if trained)
4. Repeat until help arrives or AED available
AED: Turn on, follow voice prompts
```

### Choking (Adult)
```
1. Ask "Are you choking?"
2. 5 back blows (between shoulder blades)
3. 5 abdominal thrusts (Heimlich)
4. Repeat until object expelled or unconscious
If unconscious: Start CPR (check mouth for object)
```

## Health Tracking

### Metrics to Monitor
```
Weight:        Weekly, same conditions
BP:            Weekly if elevated, monthly if normal
Resting HR:    Daily (wearable or manual)
Sleep:         Duration, quality (wearable)
HRV:           Daily trend (recovery marker)
Steps:         7,000-10,000 daily
Exercise:      Minutes, type, intensity
Mood/Energy:   Daily 1-10 scale
```

### When to See Doctor
```
New or worsening symptoms
Symptoms >2 weeks
Routine screening due
Medication refills
Vaccines due
Life changes (pregnancy, new job, stress)
Family history changes
```
