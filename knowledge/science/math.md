# Math Reference

## Arithmetic
- PEMDAS/BODMAS: Parentheses/Brackets, Exponents/Orders, Multiply/Divide (left→right), Add/Subtract
- Fractions: a/b + c/d = (ad + bc)/(bd). Multiply: a/b × c/d = ac/(bd). Divide: a/b ÷ c/d = a/b × d/c
- Percent: x% of y = (x × y)/100. x as % of y = (x/y) × 100. Change = (new-old)/old × 100%
- Ratio: a:b = c:d → ad = bc (cross multiply)

## Algebra
- ax + b = 0 → x = -b/a
- Quadratic: ax² + bx + c = 0 → x = (-b ± √(b² - 4ac)) / (2a)
  - Discriminant D = b² - 4ac: D>0 (2 real), D=0 (1 real), D<0 (2 complex)
- Slope-intercept: y = mx + b (m = slope, b = y-intercept)
- Point-slope: y - y₁ = m(x - x₁)
- Exponent rules: xᵃ·xᵇ = xᵃ⁺ᵇ, xᵃ/xᵇ = xᵃ⁻ᵇ, (xᵃ)ᵇ = xᵃᵇ, x⁰ = 1, x⁻ᵃ = 1/xᵃ
- Logarithms: logₐ(b) = c → aᶜ = b. ln(x) = logₑ(x) (natural log)
  - log(ab) = log(a) + log(b), log(a/b) = log(a) - log(b), log(aᵇ) = b·log(a)

## Geometry
- Circle: area = πr², circumference = 2πr
- Triangle: area = ½bh, sum of angles = 180°
- Rectangle: area = lw, perimeter = 2(l+w)
- Trapezoid: area = ½(b₁+b₂)h
- Sphere: surface area = 4πr², volume = ⁴⁄₃πr³
- Cylinder: volume = πr²h, surface area = 2πrh + 2πr²
- Cone: volume = ⅓πr²h
- Pythagorean: a² + b² = c² (right triangle)

## Trigonometry
- SOH CAH TOA: sin(θ) = opposite/hypotenuse, cos(θ) = adjacent/hyp, tan(θ) = opposite/adjacent
- sin²(θ) + cos²(θ) = 1
- Law of sines: a/sin(A) = b/sin(B) = c/sin(C)
- Law of cosines: c² = a² + b² - 2ab·cos(C)
- Unit circle: cos = x, sin = y. Key angles: 0°, 30°(π/6), 45°(π/4), 60°(π/3), 90°(π/2)
- Radians: 180° = π rad. To convert: deg × π/180 = rad, rad × 180/π = deg

## Statistics
- Mean (average): sum/n. Median: middle value. Mode: most frequent. Range: max-min
- Variance: avg of squared deviations from mean. Standard deviation: √variance
- Normal distribution: 68% within 1σ, 95% within 2σ, 99.7% within 3σ
- Probability: P(A) = favorable/total. P(A|B) = P(A∩B)/P(B) (conditional)
- Bayes' theorem: P(A|B) = P(B|A)·P(A)/P(B)
- Correlation r: -1 to 1 (0 = no linear relationship). Correlation ≠ causation

## Calculus
- Derivative: rate of change = slope at a point. f'(x) = limit of Δy/Δx as Δx→0
  - d/dx(xⁿ) = nxⁿ⁻¹, d/dx(eˣ) = eˣ, d/dx(ln x) = 1/x
  - d/dx(sin x) = cos x, d/dx(cos x) = -sin x
- Integral: area under curve. ∫xⁿdx = xⁿ⁺¹/(n+1) + C (n ≠ -1)
- Fundamental theorem: ∫ₐᵇ f(x)dx = F(b) - F(a) where F' = f

## Number Systems
- Binary: base 2 (0,1). Hex: base 16 (0-9, A-F)
- Powers of 2: 2¹⁰=1024 (K), 2²⁰=1,048,576 (M), 2³⁰=1,073,741,824 (G)
- Scientific notation: 1.23 × 10⁴ = 12,300. 1.23 × 10⁻⁴ = 0.000123
- Orders of magnitude:
  - nano (n) = 10⁻⁹, micro (μ) = 10⁻⁶, milli (m) = 10⁻³
  - kilo (k) = 10³, mega (M) = 10⁶, giga (G) = 10⁹, tera (T) = 10¹²

## Useful Formulas
- Distance: d = √((x₂-x₁)² + (y₂-y₁)²) (2D), same + z² (3D)
- Midpoint: ((x₁+x₂)/2, (y₁+y₂)/2)
- Simple interest: I = Prt (Principal × rate × time)
- Compound interest: A = P(1 + r/n)ⁿᵗ
- Speed: v = d/t, acceleration: a = Δv/t
- Ohm's law: V = IR. Power: P = IV = I²R = V²/R
