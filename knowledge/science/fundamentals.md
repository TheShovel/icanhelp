# Science Fundamentals

## Scientific Method

### Steps
```
1. Observation: Notice phenomenon
2. Question: "Why?" or "How?"
3. Hypothesis: Testable prediction (If...then...)
4. Experiment: Controlled test
5. Analysis: Data → patterns
6. Conclusion: Support/refute hypothesis
7. Communicate: Share, replicate
```

### Key Principles
```
Falsifiability: Must be disprovable
Reproducibility: Others get same results
Control: Isolate variables
Sample size: Large enough for significance
Peer review: Community validation
```

## Physics Basics

### Units (SI)
```
Length:        meter (m)
Mass:          kilogram (kg)
Time:          second (s)
Current:       ampere (A)
Temperature:   kelvin (K)
Amount:        mole (mol)
Luminous:      candela (cd)
```

### Derived Units
```
Force:         newton (N) = kg·m/s²
Energy:        joule (J) = N·m = kg·m²/s²
Power:         watt (W) = J/s
Pressure:      pascal (Pa) = N/m²
Charge:        coulomb (C) = A·s
Voltage:       volt (V) = J/C
Resistance:    ohm (Ω) = V/A
Frequency:     hertz (Hz) = 1/s
```

### Constants
```
c (light):           299,792,458 m/s
G (gravity):         6.674×10⁻¹¹ N·m²/kg²
h (Planck):          6.626×10⁻³⁴ J·s
k_B (Boltzmann):     1.381×10⁻²³ J/K
N_A (Avogadro):      6.022×10²³ mol⁻¹
R (gas):             8.314 J/(mol·K)
e (electron):        1.602×10⁻¹⁹ C
m_e (electron):      9.109×10⁻³¹ kg
m_p (proton):        1.673×10⁻²⁷ kg
ε₀ (permittivity):   8.854×10⁻¹² F/m
μ₀ (permeability):   4π×10⁻⁷ N/A²
```

### Mechanics
```
v = d/t                    (velocity)
a = Δv/Δt                  (acceleration)
F = ma                     (Newton's 2nd)
p = mv                     (momentum)
FΔt = Δp                   (impulse)
W = Fd cosθ                (work)
KE = ½mv²                  (kinetic energy)
PE = mgh                   (gravitational potential)
P = W/t                    (power)
τ = rF sinθ                (torque)
L = r × p                  (angular momentum)
```

### Newton's Laws
```
1. Inertia: Object at rest stays at rest, in motion stays in motion (unless force)
2. F = ma: Acceleration proportional to force, inversely to mass
3. Action-reaction: Every force has equal/opposite reaction
```

### Energy Conservation
```
Total energy constant in closed system
KE + PE + thermal + chemical + ... = constant
Efficiency = useful output / total input
```

### Thermodynamics
```
0th: Thermal equilibrium transitive
1st: ΔU = Q - W (energy conserved)
2nd: Entropy increases (disorder)
    Heat flows hot → cold
    No 100% efficient engine
3rd: Absolute zero unattainable
    Entropy → 0 as T → 0 K
```

### Waves
```
v = fλ                    (speed = frequency × wavelength)
f = 1/T                   (frequency = 1/period)
E = hf                    (photon energy)
c = fλ                    (light)
Snell: n₁sinθ₁ = n₂sinθ₂  (refraction)
Doppler: f' = f(v±v_o)/(v∓v_s)
```

## Chemistry Basics

### Atomic Structure
```
Proton:  +1 charge, ~1 amu, in nucleus
Neutron: 0 charge, ~1 amu, in nucleus
Electron: -1 charge, ~1/1836 amu, orbitals
Atomic number = protons
Mass number = protons + neutrons
Isotopes: Same element, different neutrons
```

### Periodic Table
```
Groups (columns): Same valence electrons → similar properties
Periods (rows):   Increasing atomic number
Metals: Left, lose electrons, conductors
Nonmetals: Right, gain/share electrons
Metalloids: Staircase line
```

### Bonding
```
Ionic:    Metal + nonmetal, electron transfer, crystal lattice (NaCl)
Covalent: Nonmetal + nonmetal, electron sharing (H₂O, CO₂)
  Polar:  Unequal sharing (electronegativity diff 0.4-1.7)
  Nonpolar: Equal sharing (<0.4)
Metallic: Metal + metal, electron sea
Hydrogen bond: H bonded to N,O,F attracted to N,O,F (water properties)
```

### Mole Concept
```
1 mole = 6.022×10²³ particles (Avogadro's number)
Molar mass = atomic mass in g/mol
n = m/M (moles = mass / molar mass)
M = n/V (molarity = moles / liters)
```

### Stoichiometry
```
1. Balance equation
2. Convert given → moles
3. Mole ratio from coefficients
4. Convert moles → desired unit
Limiting reagent: Runs out first, determines product
Percent yield = actual/theoretical × 100%
```

### Solutions
```
Molarity (M) = mol solute / L solution
Molality (m) = mol solute / kg solvent
Dilution: M₁V₁ = M₂V₂
Colligative: Depend on solute particles
  ΔTf = i·Kf·m (freezing point depression)
  ΔTb = i·Kb·m (boiling point elevation)
  π = iMRT (osmotic pressure)
```

### Acids & Bases
```
Arrhenius: Acid → H⁺, Base → OH⁻
Brønsted-Lowry: Acid = proton donor, Base = proton acceptor
Lewis: Acid = electron pair acceptor, Base = donor
pH = -log[H⁺]
pOH = -log[OH⁻]
pH + pOH = 14 (25°C)
Kw = [H⁺][OH⁻] = 1×10⁻¹⁴
Strong: HCl, HNO₃, H₂SO₄, NaOH, KOH
Weak: CH₃COOH, NH₃, HCN
Buffer: Weak acid + conjugate base (resists pH change)
```

### Redox
```
Oxidation: Loss of electrons (OIL)
Reduction: Gain of electrons (RIG)
Oxidizing agent: Gets reduced
Reducing agent: Gets oxidized
Assign oxidation numbers
Balance half-reactions
```

## Biology Basics

### Cell Theory
```
1. All organisms made of cells
2. Cell = basic unit of life
3. Cells come from pre-existing cells
```

### Cell Types
```
Prokaryote: No nucleus, no organelles (bacteria, archaea)
Eukaryote: Nucleus, organelles (plants, animals, fungi, protists)
```

### Organelles
```
Nucleus:      DNA, control center
Mitochondria: ATP production (cellular respiration)
Ribosomes:    Protein synthesis
ER:           Rough (ribosomes) = protein processing; Smooth = lipids
Golgi:        Package, modify, ship proteins
Lysosomes:    Digestion (enzymes)
Vacuole:      Storage, turgor (plants)
Chloroplast:  Photosynthesis (plants)
Cell membrane: Phospholipid bilayer, selective permeability
```

### Central Dogma
```
DNA → RNA → Protein
Transcription: DNA → mRNA (nucleus)
Translation: mRNA → protein (ribosome)
Codon: 3 bases = 1 amino acid
Genetic code: Universal (mostly)
```

### Genetics
```
Gene: DNA segment coding for protein
Allele: Gene variant
Dominant: Expressed in heterozygote
Recessive: Only expressed in homozygote
Genotype: Allele combination
Phenotype: Observable trait
Punnett square: Predict offspring ratios
Mendel's laws: Segregation, Independent assortment
```

### Evolution
```
Natural selection: Variation + Heritability + Differential reproduction
Evidence: Fossils, comparative anatomy, embryology, DNA, biogeography
Speciation: Reproductive isolation → new species
Phylogeny: Evolutionary relationships (tree of life)
```

## Earth Science

### Earth Structure
```
Crust: 5-70 km (oceanic thinner, continental thicker)
Mantle: 2900 km, solid but flows
Outer core: Liquid iron/nickel
Inner core: Solid iron/nickel
```

### Plate Tectonics
```
Divergent: Plates apart → mid-ocean ridges, rift valleys
Convergent: Plates together → subduction, mountains, volcanoes
Transform: Plates slide past → earthquakes (San Andreas)
Hotspots: Mantle plumes → volcanic islands (Hawaii)
```

### Rock Cycle
```
Igneous: Cooled magma/lava (granite, basalt)
Sedimentary: Compacted sediment (sandstone, limestone)
Metamorphic: Heat/pressure changed (marble, slate)
Weathering → Erosion → Deposition → Compaction → Metamorphism → Melting
```

### Water Cycle
```
Evaporation → Condensation → Precipitation → Infiltration/Runoff → Evaporation
Groundwater: Aquifers, water table
```

### Atmosphere Layers
```
Troposphere: 0-12 km (weather, life)
Stratosphere: 12-50 km (ozone layer)
Mesosphere: 50-85 km (meteors burn)
Thermosphere: 85-600 km (aurora, ISS)
Exosphere: 600+ km (satellites)
```

### Climate
```
Weather: Short-term atmospheric conditions
Climate: Long-term patterns (30+ years)
Greenhouse effect: Gases trap IR radiation
CO₂, CH₄, N₂O, H₂O = main greenhouse gases
```

```

Planets: Mercury, Venus, Earth, Mars (terrestrial)
         Jupiter, Saturn (gas giants)
         Uranus, Neptune (ice giants)
Dwarf planets: Pluto, Ceres, Eris
Asteroid belt: Mars-Jupiter
Kuiper belt: Beyond Neptune
Oort cloud: Comet reservoir
```

### Stars
```
Nebula → Protostar → Main sequence → Red giant → White dwarf/Neutron star/Black hole
Mass determines fate
Main sequence: H → He fusion
Luminosity ∝ Mass³·⁵
```

### Distances
```
AU: Earth-Sun = 150 million km
Light-year: 9.46 trillion km
Parsec: 3.26 ly = 206,265 AU
Nearest star: Proxima Centauri (4.24 ly)
```

## Measurement & Data

### Significant Figures
```
Non-zero: significant
Zeros between: significant
Leading zeros: not significant
Trailing zeros with decimal: significant
Trailing zeros without decimal: ambiguous
Multiplication/division: least sig figs
Addition/subtraction: least decimal places
```

### Uncertainty
```
Absolute: ± value
Relative: absolute / measured
Percent: relative × 100%
Propagation: Add absolute for +/-, add relative for ×/÷
```

### Statistics
```
Mean: Σx/n
Median: Middle value
Mode: Most frequent
Std dev: √(Σ(x-μ)²/n) (population) or n-1 (sample)
Variance: σ²
Standard error: σ/√n
Confidence interval: mean ± z·SE
```

### Graphing
```
Independent variable: x-axis
Dependent variable: y-axis
Linear: y = mx + b
Slope: Δy/Δx
Intercept: y at x=0
R²: Goodness of fit (0-1)
Log scale: For exponential/power relationships
```

## Safety & Lab Practices

### General
```
Know hazards (SDS sheets)
PPE: Goggles, gloves, lab coat, closed shoes
No food/drink
Label everything
Know exits, eyewash, shower, extinguisher
```

### Chemical
```
Add acid to water (not water to acid)
Fume hood for volatiles
Compatible storage (acids separate from bases, oxidizers from flammables)
Waste: Separate by hazard class
```

### Biological
```
BSL-1: Non-pathogenic, standard microbiology
BSL-2: Moderate hazard (hepatitis, salmonella)
BSL-3: Serious airborne (TB)
BSL-4: Deadly, no treatment (Ebola)
```

### Radiation
```
Time, Distance, Shielding
ALARA: As Low As Reasonably Achievable
α: Paper stops
β: Plastic/aluminum stops
γ: Lead/concrete needed
```
